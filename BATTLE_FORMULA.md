# 전투 공식 상세 (확정)

> 모든 전투 계산의 기준 문서
> 마지막 업데이트: 2026-04-11

---

## 1. 데미지 계산

```
보호막 있으면: ATK → 보호막 먼저 흡수 (DEF/MEVA 무관) → 초과분에 DEF/MEVA 적용
보호막 없으면: 바로 DEF/MEVA 적용

물리 데미지 = ATK - DEF (최소 0)
마법 데미지 = ATK - MEVA (최소 0)
  → 마법공격은 DEF 완전 무시! MEVA로만 방어
  → 보호막은 DEF/MEVA 영향 없이 ATK 원본 흡수

크리티컬 시 = 데미지 × 2
원소 유리 시 = 데미지 × 1.5
원소 불리 시 = 데미지 × 0.75
```

> 공격 유형은 스펠에 명시: "물리공격" 또는 "마법공격"
> 일반 공격(기본 공격)은 항상 물리
> 마법공격은 DEF 무시 — MEVA로만 방어 가능
> 보호막은 모든 데미지 흡수 (방어력과 무관한 별도 층)

---

## 2. 행동 순서

```
1순위: 특성 (선제공격 등)
2순위: SPD 높은 순
3순위: 동일 SPD → 랜덤
```

---

## 3. 크리티컬 (급소)

```
크리티컬 확률 = LUCK × 10%

예시:
  LUCK 1 = 10%
  LUCK 3 = 30%
  LUCK 5 = 50%
  LUCK 10 = 100% (항상 크리티컬)

크리티컬 데미지 = 기본 데미지 × 2
```

---

## 4. 회피 (EVA)

```
회피 확률 = EVA × 10%

예시:
  EVA 1 = 10%
  EVA 3 = 30%
  EVA 5 = 50%

규칙:
  - 물리 공격만 회피 가능
  - 마법 공격은 회피 불가
  - 마법 회피는 유물/장비 특수효과로만 가능
  - 회피 성공 시 데미지 0 (완전 회피)
```

---

## 5. 마법방어 (MEVA)

```
마법 데미지 = ATK - MEVA (최소 0)

적용 대상:
  - 스펠 중 "마법공격"으로 분류된 것만
  - 일반 공격(기본 공격)에는 DEF 적용
  - 물리 스펠에는 DEF 적용

공격 유형 구분:
  물리공격 → DEF로 방어, EVA로 회피 가능
  마법공격 → MEVA로 방어, 회피 불가 (유물/장비 제외)
```

---

## 6. 에너지 (NRG) & 스펠 코스트

```
스펠 시전 조건: 현재 NRG >= 스펠 코스트
시전 시: NRG -= 스펠 코스트
NRG 부족하면: 스펠 사용 불가 (일반 공격만)
  → UI 경고: "⚡ 에너지 부족! (필요: X / 현재: Y)" 표시

NRG 회복: 매턴 종료 시 nrgReg만큼 회복
  기본 nrgReg = 1
  지원 유닛 nrgReg = 2

특수 코스트:
  일부 스펠은 NRG 대신 HP를 소모 (예: 피의갈증)
  HP 코스트 스펠은 NRG 0이어도 사용 가능
```

---

## 7. 분노 (RAGE)

```
분노 축적: 피격 시 받은 데미지만큼 RAGE 증가
  예: 5 데미지 받으면 RAGE +5

분노 단계별 효과:
  RAGE 10 도달 → 공격력 +10%
  RAGE 20 도달 → 공격력 +20%
  RAGE 30 도달 → 공격력 +30%
  RAGE 100 도달 → 궁극기 발동!
    - 보유 스펠 중 코스트가 가장 높은 스펠을 NRG 소모 없이 즉시 시전
    - 발동 후 RAGE 0으로 리셋
    - 다시 축적 시작

분노 효과 중첩:
  RAGE 25이면 → 20 단계 효과 적용 (공격력 +20%)
  RAGE 35이면 → 30 단계 효과 적용 (공격력 +30%)
  RAGE 100이면 → 궁극기 + 리셋
```

---

## 8. 체력 회복 (hpReg)

```
회복 타이밍: 매턴 종료 후 자동 회복
회복량: hpReg 수치만큼 HP 회복 (최대 HP 초과 불가)

기본값:
  근접 전사 영웅 hpReg = 2 (2차이상은 추후에)
  그 외 모든 유닛 hpReg = 1
```

---

## 9. 턴 진행 순서 (요약)

```
1. 턴 시작
   └─ 상태이상 체크 (빙결=행동불가, 화상=데미지)

2. 행동 순서 결정
   └─ 특성 > SPD > 랜덤

3. 각 유닛 행동 (순서대로)
   ├─ NRG 충분하면 → 스펠 시전 (마법/물리)
   └─ NRG 부족하면 → 경고 "⚡ 에너지 부족!" + 일반 공격 (물리)

4. 데미지 판정
   ├─ 물리: ATK - DEF, 회피 판정(EVA×10%)
   ├─ 마법: ATK - MEVA, 회피 불가
   └─ 크리티컬 판정 (LUCK×10%, 성공시 ×3)

5. 피격 처리
   ├─ HP 감소
   ├─ RAGE 증가 (받은 데미지만큼)
   └─ RAGE 단계 체크 (10/20/30/100)

6. 사망 체크
   ├─ HP 0 → 사망 (부활 키워드 있으면 부활)
   └─ 영웅 사망 → 전투 종료

7. 턴 종료
   ├─ HP 자동 회복 (hpReg)
   ├─ NRG 자동 회복 (nrgReg)
   └─ 상태이상 턴 수 감소

8. 다음 턴 → 1번으로
```

## 10. 상태이상 정리

| 상태이상 | 효과 | 지속 | 방어 가능 |
|---------|------|------|----------|
| 화상 | 턴 시작 시 고정 데미지 | N턴 | MEVA로 감소 |
| 빙결 | 1턴 행동 불가 | 1턴 | - |
| 독 | 턴 시작 시 2 데미지 | N턴 | - |
| 동상 | SPD -3 | N턴 | - |
| 은신 | 다음 턴 회피 100% | 1턴 | - |
| 보호막 | 데미지 흡수 (수치만큼) | 소진까지 | - |
