# 키워드 발동 순서 & 콤보 시스템

> 전투 중 키워드/특성이 동시에 발동할 때의 우선순위
> 마지막 업데이트: 2026-04-11

---

## 1. 키워드 발동 순서 (Phase 순)

### Phase 0: 턴 시작 (상태이상 처리)
```
화상 데미지 적용
독 데미지 적용
빙결 체크 (행동 불가)
동상 체크 (SPD 감소)
상태이상 턴 수 -1
```

### Phase 1: 타겟 결정
```
1-1. 도발 → 도발 유닛이 있으면 강제 타겟
1-2. 은신 → 은신 유닛은 타겟 대상에서 제외
```

### Phase 2: 행동 순서 결정
```
2-1. 선제공격 → 무조건 최우선 행동
2-2. SPD 비교 → 높은 순
2-3. 동일 SPD → 랜덤
```

### Phase 3: 공격 판정
```
3-1. 관통 → DEF 무시 (물리만)
3-2. 회피 판정 → EVA×10% (물리만, 마법 회피 불가)
3-3. 크리티컬 판정 → LUCK×10%, 성공 시 데미지 ×2
3-4. 데미지 계산 → ATK - DEF (or ATK - MEVA)
3-5. 원소 상성 적용 → ×1.5 유리 / ×0.75 불리
```

### Phase 4: 피격 후 효과
```
4-1. 흡혈 → 가한 데미지만큼 HP 회복
4-2. 반격 → 받은 데미지의 50% 되돌림
4-3. 가시 → 공격자에게 고정 데미지
4-4. 화염오라 → 공격자에게 화염 데미지
4-5. RAGE 축적 → 받은 데미지만큼 분노 증가
4-6. RAGE 단계 체크 → 10/20/30 ATK%, 100 궁극기
```

### Phase 5: 처치 후 효과
```
5-1. 영혼수확 → 처치 시 ATK +3 영구
5-2. 은신 → 처치 시 다음 턴 회피 100%
5-3. 사령술 → 처치한 적을 아군으로 소환
5-4. 울부짖음 → 아군 야수 ATK +1
```

### Phase 5.5: 보호막 판정 (DEF/MEVA 계산 전)
```
보호막 규칙:
  - DEF/MEVA 영향 안 받음 (ATK 원본이 그대로 보호막에 적용)
  - 보호막 수치만큼 흡수 → 초과분에 DEF/MEVA 적용 → HP 감소
  - 물리/마법 모든 데미지 흡수

  계산 순서:
    1. ATK 10 → 보호막 3 흡수 → 남은 데미지 7
    2. 남은 7 → DEF 5 적용 → HP에 2 데미지

  보호막은 중첩 가능 (여러 소스에서 부여 시 합산)
  턴 제한 없음 (소진될 때까지 유지)
```

### Phase 6: 사망 판정
```
6-1. 부활 → 첫 사망 시 HP 50%로 부활
6-2. 영웅 사망 → 전투 종료
```

### Phase 7: 턴 종료
```
7-1. HP 자동 회복 (hpReg)
7-2. NRG 자동 회복 (nrgReg)
7-3. 버프/디버프 턴 수 감소
```

## 2. 콤보 시스템

### 연계 공격 (Chain Attack)
```
발동 조건: 같은 종족 유닛이 2체 이상 있을 때
확률: 종족 수 × 10%
효과: 공격 직후, 같은 종족 아군 1체가 추가 공격

예시:
  늑대 A가 공격 → 30% 확률로 늑대 B가 추가 공격!
  (야수 3마리 = 30%)

UI 연출:
  "🐺 연계 공격!" 배너 표시
  추가 공격 유닛이 빠르게 돌진하는 이펙트
```

### 협동 스킬 (Assist Skill)
```
발동 조건: 같은 원소 유닛이 2체 이상 있을 때
확률: 원소 수 × 15%
효과: 스펠 시전 시, 같은 원소 아군이 보조 효과 추가

예시:
  화염술사가 화염폭발 시전 →
  불꽃정령이 협동 → 화상 1턴 추가!

조합별 협동 효과:
  🔥+🔥 → 화상 턴 수 +1
  💧+💧 → 빙결 확률 +15%
  ⚡+⚡ → 추가 타격 1회
  🌿+🌿 → 대상 SPD -2
  🌑+🌑 → 흡혈 효과 +10%
  ✨+✨ → 치유량 +30%
```

### 수호 (Protect)
```
발동 조건: 방어 역할(defense) 유닛이 있을 때
확률: 25%
효과: 아군이 치명상(HP 20% 이하)을 받을 때,
      방어 유닛이 대신 피격

예시:
  적이 마법사를 공격 → 기사가 앞으로 뛰어나와 대신 맞음!
  "🛡️ 수호!" 배너

UI 연출:
  방어 유닛이 순간이동하여 아군 앞에 서는 이펙트
  방패 빛남 + 충격파
```

### 분노 폭발 (Rage Burst)
```
발동 조건: RAGE 100 도달 시
효과: 가장 코스트 높은 스펠을 NRG 무소모 즉시 시전
      + 주변 아군 RAGE +10 전파

UI 연출:
  화면 흔들림 + 붉은 오라 폭발
  "💢 분노 폭발!" 배너
  슬로모션 0.5초 → 스펠 시전
```

### 연속기 (Combo Strike)
```
발동 조건: 같은 유닛이 2턴 연속 같은 적을 공격할 때
효과: 2턴째 데미지 +20%
      3턴 연속 시 데미지 +40% + 방어력 관통

예시:
  암살자가 적 기사를 3턴 연속 공격 →
  1턴: 일반 데미지
  2턴: +20% "연속기!"
  3턴: +40% + 관통 "완벽한 연속기!!"

UI 연출:
  2연속: "⚔️ 연속기!" (노란색)
  3연속: "⚔️⚔️ 완벽한 연속기!!" (빨간색 + 화면 흔들림)
```

### 최후의 일격 (Final Strike)
```
발동 조건: 아군 영웅 HP가 10% 이하일 때
효과: 영웅의 ATK +50%, 모든 스펠 코스트 -50%
      (죽기 직전의 각성)

UI 연출:
  화면 어두워짐 + 영웅에게 스포트라이트
  심장박동 소리 + 붉은 오라
  "💀 최후의 일격..." 배너
```

### 동료의 복수 (Avenger)
```
발동 조건: 아군 유닛이 사망했을 때
효과: 사망한 유닛과 같은 종족 아군의 ATK +20% (3턴)

예시:
  늑대 A가 사망 → 늑대 B, C의 ATK +20%!
  "🔥 동료의 복수!" 배너

UI 연출:
  사망한 유닛의 잔상이 아군에게 흡수되는 이펙트
  아군 테두리 빨간색 빛남
```

| 협동 스킬 | 같은 원소 2체+ | 원소수×15% | 스펠 보조 효과 |
