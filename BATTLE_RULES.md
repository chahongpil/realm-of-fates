## 0. 매칭 대기 화면 — 상대 정보 공개 (신규, 2026-04-12)

오토배틀러는 "상대 대응 전략"이 재미의 핵심이므로, 매칭 성사 후 **10초 대기 시간** 동안 상대 정보를 공개한다.

### 공개 항목
```
╔═ 상대와 곧 만납니다 (10초) ═══════╗
║                                  ║
║  🛡️ [영웅 초상]  Lv.12           ║
║     "불의 대행자 카일라스"        ║
║     원소: 🔥 화염  | 성향: 공격   ║
║                                  ║
║ 📜 보유 유닛 (5체):               ║
║   🔥 화염술사 Lv3  🐺 늑대 Lv5   ║
║   🗡️ 민병 Lv2     ⚔️ 기사 Lv4    ║
║   🦅 썬더버드 Lv3                ║
║                                  ║
║ ⚡ 활성 시너지:                   ║
║   🔥 화염 x2 (발동)              ║
║   👤 인간 x3 (발동)              ║
║                                  ║
║ 🏆 최근 전적: 3연승 중 (주의!)    ║
║                                  ║
║ [전투 시작]    ⏱ 7초 남음         ║
╚══════════════════════════════════╝
```

### 대응 전략 유도
- 플레이어는 이 10초 동안 **편성 최종 변경** 가능
- 예: "상대 화염 특화" → 물 유닛 앞열 배치
- 예: "3연승 중 상대" → 경계하여 보호막 유물 장착
- **자동 스킵 옵션**: 숙련자용 (설정에서 on/off)

### 왜 공개하는가?
- 정보 불투명 = 도박. 정보 투명 = 전략.
- TFT 계보 오토배틀러는 모두 상대 정보 공개 → 대응 전략이 재미의 핵심.

## 1. 라운드 선택지 (확정)

### 구조
```
전투 시작
  ↓
라운드 1 (1회 공격 교환)
  ↓
선택지 화면 (행동력 AP만큼 선택 가능)
  ↓
라운드 2 (2회 공격 교환)
  ↓
선택지 화면
  ↓
... 반복 (영웅 사망 시 종료)
```

### 행동력 (AP)
| 영웅 레벨 | AP | 의미 |
|----------|-----|------|
| Lv1 | 1 | 1번만 선택 |
| Lv2~9 | 2 | 2번 선택 |
| Lv10~49 | 3 | 3번 선택 |
| Lv50~100 | 4 | 4번 선택 |

### 선택지 5가지 (구현 정본)

| 선택 | 빈 슬롯 있을 때 | 슬롯 꽉 찼을 때 |
|------|---------------|---------------|
| **🗡️ 유닛** | 새 유닛 소환 | 기존 유닛 강화 (스탯 +1~3 랜덤) |
| **⚡ 스킬** (스펠 카드) | 새 스킬 습득 → 유닛에 장착 | 기존 스킬 강화 (효과 +50%) |
| **🏺 유물** | 새 유물 획득 → 덱 전체 버프 | 기존 유물 강화 (효과 수치 업) |
| **🧪 포션** | 영웅 HP 50% 회복 (쿨다운 1라운드) | 동일 |
| **⏭️ 턴 종료** | 다음 라운드 (무료) | 동일 |

> ※ 포션 = 영웅 HP 50% 회복 (유닛 전체 회복 아님). AI는 HP 30% 이하일 때 자동 사용.

### 슬롯 제한
| 종류 | 최대 | 꽉 차면 |
|------|------|--------|
| 유닛 | 5체 | 강화만 가능 |
| 스펠 | 유닛당 3개 | 해당 유닛 강화만 |
| 유물 | 5개 | 강화만 가능 |

### 선택지 화면 UI
```
╔═ 라운드 3 종료 — 다음 전략을 선택하세요 ═╗
║                                          ║
║  [🗡️ 유닛]  [⚡ 스펠]  [🏺 유물]         ║
║                                          ║
║  [💤 휴식]  [⏭️ 턴 종료]                 ║
║                                          ║
║  남은 행동력: 🟡🟡⚫ (2/3)               ║
╚══════════════════════════════════════════╝

유닛 선택 시:
┌──────────────────────────────────┐
│ 3장의 유닛 카드 제시 (등급 랜덤)  │
│                                  │
│ [카드A]  [카드B]  [카드C]         │
│                                  │
│ [🔄 리롤 2💰]  [← 돌아가기]      │
└──────────────────────────────────┘
```

### 등급 확률 (라운드에 따라 상승)
| 라운드 | 브론즈 | 실버 | 골드 | 전설 | 신 |
|--------|--------|------|------|------|---|
| 1~3 | 70% | 25% | 5% | 0% | 0% |
| 4~6 | 50% | 30% | 15% | 5% | 0% |
| 7~9 | 30% | 30% | 25% | 13% | 2% |
| 10+ | 20% | 25% | 30% | 20% | 5% |

## 2. AI 상대 로직

### AI 판단 우선순위
```
1. 영웅 HP 30% 이하 → 휴식 선택 (최우선)
2. 유닛 슬롯 비어있음 → 유닛 소환
3. 유닛에 스펠 빈 슬롯 → 스펠 습득 (50% 확률)
4. 유물 슬롯 비어있음 → 유물 획득 (30% 확률)
5. 유닛 슬롯 꽉 참 → 가장 약한 유닛 강화
6. 위 해당 없음 → 턴 종료
```

### AI 난이도 (리그별)
| 리그 | AI 특징 |
|------|--------|
| 브론즈 | 랜덤 선택, 실수 많음 (30% 비최적 선택) |
| 실버 | 기본 우선순위 따름 |
| 골드 | 원소 상성 고려, 시너지 편성 |
| 플래티넘 | 스펠 코스트 최적화, 콤보 활용 |
| 다이아+ | 플레이어 편성 분석, 카운터 편성 |

### AI 전투 중 스펠 사용
```
1. NRG 충분 + 가장 효율적 스펠 → 시전
2. 영웅 HP 위험 + 치유 스펠 보유 → 치유 우선
3. 적 HP 낮음 + 공격 스펠 → 마무리 시도
4. NRG 부족 → 일반 공격
```

### AI 타겟 선택
```
1. 도발 유닛 있으면 → 강제 타겟
2. 적 영웅 HP 20% 이하 → 영웅 집중 (60% 확률)
3. 적 힐러/지원 유닛 → 우선 제거 (40% 확률)
4. 가장 HP 낮은 적 → 마무리 (30% 확률)
5. 랜덤 타겟
```

## 3. 덱 편성 규칙

### 편성 구조
```
전투 출전 편성:
  영웅 1체 (필수, 중앙 고정)
  + 동료 유닛 최대 4체 (선택)
  = 최대 5체, 가로 1열 배치
```

### 배치 규칙
```
영웅 항상 중앙 고정, 중앙부터 채워나감:
  1체: [  ] [  ] [영웅] [  ] [  ]
  2체: [  ] [유닛] [영웅] [  ] [  ]
  3체: [  ] [유닛] [영웅] [유닛] [  ]
  4체: [유닛] [유닛] [영웅] [유닛] [  ]
  5체: [유닛] [유닛] [영웅] [유닛] [유닛]

- 드래그로 위치 교환 가능
- 부상 유닛은 출전 불가 (교회에서 치유 필요)
- 같은 유닛 중복 출전 불가
- 장비/스펠은 편성 전에 장착해야 함
```

### 전투 화면 레이아웃

#### 1. 평소 (오토배틀 진행 중)
```
┌─────────────────────────────────────┐
│ [라운드 R3]                  [⏩×2] │
├─────────────────────────────────────┤
│  [적1] [적2] [적영웅] [적3] [적4]   │
│ ════════════════════════════════════ │
│  [유닛1] [유닛2] [영웅⭐] [유닛3] [유닛4] │
│                                     │
│  [스킬바: 시전 가능 유닛들]          │
├─────────────────────────────────────┤

└─────────────────────────────────────┘
  HP/NRG는 각 카드에 숫자로 표시 (별도 바 없음)
```

#### 2. 스킬바에서 유닛 탭 → 유닛 카드가 정면에 올라옴
```
┌─────────────────────────────────────┐

│              [← 취소]              │

```

#### 3. 스펠 탭 → 적 선택 화면 (타겟 지정)
```

│  [적1] [적2] [적영웅] [적3] [적4]   │ ← 적 다시 나타남

```

## 4. 유닛 사망/부상 시스템

### 전투 중 사망
```
유닛 HP 0 → 전투에서 퇴장
  - 부활 키워드 있으면 → HP 50%로 부활 (1회)
  - 없으면 → 해당 전투에서 사용 불가

영웅 HP 0 → 전투 패배 (즉시 종료)
```

### 전투 후 처리
```
승리 시:
  - 생존 유닛: 정상 (HP 전투 전으로 복구)
  - 전투 중 사망한 유닛: "부상" 상태

패배 시:
  - 영웅: 자동 부활 (마을로 귀환)
  - 전투 중 사망한 유닛: "부상" 상태
  - 생존 유닛: 정상
```

### 부상 시스템
```
부상 상태:
  - 출전 불가 (치유 전까지)
  - 카드에 🩹 표시
  - 교회에서 치유 or 장례

교회 치유:
  - 비용: 유닛레벨 × 2💰
  - 즉시 치유 → 출전 가능

교회 장례:
  - 유닛 영구 삭제
  - 보상: 명예 + 소량 골드
  - 돌이킬 수 없음 (부활권 ✨3으로 복구 가능)
```

### 영구 사망은 없음
```
- 유닛은 부상만 됨 (영구 사망 X)
- 장례는 플레이어가 직접 선택해야만 삭제
- 실수 방지: "정말 장례를 치르시겠습니까?" 2중 확인
```

## 5. 튜토리얼 흐름 (신규 유저 첫 5분)

### Step 0: 프롤로그 (30초)
```
어두운 화면 + 텍스트 타이핑 연출:
  "태초에 6원소의 신이 있었다..."
  "신들은 직접 싸울 수 없었기에..."
  "인간 영웅을 대리인으로 선택했다..."
  "당신이 그 선택받은 자다."

[운명을 받아들인다] [건너뛰기]
```

### Step 1: 영웅 선택 (1분)
```
"당신의 원소를 선택하세요"
  🔥 불 / 💧 물 / ⚡ 번개 / 🌿 대지 / 🌑 암흑 / ✨ 신성

"당신의 전투 방식을 선택하세요"
  ⚔️ 근접 전사 / 🏹 원거리 궁수 / 🔮 지원 마법사

→ 영웅 확정 + 동료 1체 자동 지급
```

### Step 2: 마을 첫 방문 (1분)
```
카메라가 폐허 마을을 보여줌 (빈 터만)

NPC 동료 등장 (말풍선):
  "영웅님, 여기가 우리 마을이 될 곳이에요!"
  "먼저 성문을 세워서 전투에 나갈 준비를 하죠."

→ 성문 건설 유도 (무료)
→ 건설 뿅! 애니메이션

  "이제 서고를 세워서 전력을 확인해보세요."

→ 서고 건설 유도 (5💰, 초기 골드 5 = 즉시 건설 가능 / Phase1 5-1 확정)
```

### Step 3: 첫 전투 (2분)
```
NPC:
  "성문이 열렸어요! 첫 전투에 나가볼까요?"

→ 편성 화면 (영웅 + 동료 1체 자동 배치)
→ 편성 설명 하이라이트

전투 시작:
  - 상대: AI 브론즈 (매우 약함, 패배 불가 수준)
  - 1라운드 자동 진행 + 설명 말풍선

라운드 사이 선택:
  NPC: "유닛을 소환해서 동료를 늘려보세요!"
  → 유닛 선택 강제 안내

2라운드 진행:
  NPC: "스펠도 사용할 수 있어요! 부채꼴에서 골라보세요."

→ 승리 (확정)
→ 보상 화면 설명
```

### Step 4: 마을 발전 (30초)
```
NPC:
  "축하해요! 이제 선술집을 세워서 더 강한 동료를 모아보세요."

→ 선술집 건설 유도 (10💰)
→ 선술집 주모 NPC 첫 등장

  "이제 자유롭게 모험하세요! 리그에서 더 높이 올라가봐요!"

→ 튜토리얼 종료
→ 자유 플레이
```

### 튜토리얼 체크포인트
| Step | 트리거 | 완료 조건 |
|------|--------|----------|
| 0 | 첫 로그인 | 프롤로그 완료 |
| 1 | 프롤로그 후 | 영웅 선택 완료 |
| 2 | 영웅 선택 후 | 성문+서고 건설 |
| 3 | 건설 후 | 첫 전투 승리 |
| 4 | 첫 전투 후 | 선술집 건설 |
| 완료 | 선술집 후 | 자유 플레이 |

### 튜토리얼 이후 자연스러운 유도
```
- 첫 패배 시: "교회를 세우면 부상 동료를 치유할 수 있어요"
- 유닛 3체 이상: "대장간을 세워서 장비를 만들어보세요"
- Lv5 도달: "성을 증축하면 더 강해질 수 있어요"
- 리그 실버 진입: "훈련장에서 수련할 수 있어요"
```
