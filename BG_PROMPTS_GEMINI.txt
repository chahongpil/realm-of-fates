📐 공통 출력 규격
─────────────────────────────────────────────────
- 크기: 1920 x 1080 (16:9 가로, Full HD)
- 게임 실제 뷰포트: 1280x720, CSS 는 center/cover 로 배치
- 파일: PNG, c:\work\game\img\ 에 저장 (기존 파일 덮어쓰기)
- 가장자리 어둡게 (비네트) — UI 가 올라가는 영역 가독성 확보
- 중앙 1280x720 범위는 "UI 가 들어올 구역" → 과한 디테일/얼굴 금지
- 텍스트·로고·워터마크 금지

4. c:\work\game\img\ 에 덮어쓰기
5. 브라우저 Ctrl+F5
