## 1. RAGE 100 + 스펠 0개 → 크래시 방지

```
문제: RAGE 100 도달 시 "가장 코스트 높은 스펠 시전"인데
      스펠이 0개면 null 참조 → 크래시

해결:
  if (스펠 0개) → ATK ×2 강화 일반공격으로 대체
  RAGE 리셋은 동일 (100 → 0)
```

## 2. 보호막 + 가시 + 흡혈 동시 발생 순서

```
문제: 공격 시 보호막/가시/흡혈이 동시에 발동하면 순서 충돌

확정 순서:
  1. 공격자 ATK 결정
  2. 보호막 먼저 흡수 (DEF 무관, ATK 원본)
  3. 보호막 초과분 → DEF/MEVA 적용 → 실제 HP 데미지
  4. 흡혈 = 실제 HP 데미지 기준으로만 회복 (보호막 흡수분 제외)
  5. 가시/반격 = 실제 HP 데미지 기준으로 반사
  6. RAGE 축적 = 실제 HP 데미지만큼

핵심: "실제 HP에 들어간 데미지"만이 흡혈/가시/RAGE의 기준
      보호막이 흡수한 데미지는 계산에서 제외
```

## 3. 회피율(EVA) 최대 캡

```
문제: 절대회피 장비 + EVA 스탯 투자 → 100% 회피 → 무적

해결:
  EVA 최대 캡 = 80% (어떤 조합이든 80% 초과 불가)
  절대회피 장비: 마법도 회피 가능하지만 캡은 동일 80%
  은신(100% 회피)은 1턴 한정이므로 예외 허용

  계산: min(EVA × 10%, 80%)
```

## 4. NRG 상한선

```
문제: NRG에 상한이 없으면 무한 축적 → 고코스트 스펠 연발

해결:
  NRG 최대 = 기본 NRG × 2

  예시:
    지원 마법사 기본 NRG 30 → 최대 60
    근접 전사 기본 NRG 5 → 최대 10

  Lv100 올스탯 NRG 투자 시: 기본 30 + (99×5) = 525 → 최대 1050
  → 이건 너무 크니까... 

  대안: NRG 최대 = 기본NRG + (레벨 × 5) + 장비보너스, 캡 200
  → NRG 캡 = 200 (어떤 조합이든 200 초과 불가)
```

## 5. 흡혈 무한 힐 루프 방지

```
문제: 흡혈 유닛이 보호막 있는 적을 때리면
      보호막만 깎이고 HP 안 줄어드는데 흡혈은 발동?

해결: (2번에서 확정)
  흡혈 = 실제 HP 데미지 기준
  보호막만 깎힌 경우 → 흡혈 0 (HP 데미지 없으니까)

  추가 규칙:
  - 같은 턴에 같은 유닛을 여러 번 치유 불가 (중복 힐 방지)
  - hpReg + 흡혈 + 스펠 치유 = 전부 합산 후 1회 적용
```

## 6. 영웅 혼자 출전 (유닛 0체)

```
문제: 유닛 없이 영웅만 출전 → 타겟 없는 스킬 오류

해결:
  - 영웅 혼자 출전 허용 (첫 전투에서 발생 가능)
  - 아군 대상 스킬 (치유 등) → 영웅 자신에게 시전
  - "같은 종족 아군" 참조 스킬 → 대상 없으면 스킵 (에러 안 남)
  - 울부짖음 등 → 대상 0이면 발동만 하고 효과 없음
```

## 7. ATK 버프 중첩 계산 순서

```
문제: 기본ATK + 장비ATK + 영혼수확(+2/킬) + RAGE% 가 동시에 있으면
      어떤 순서로 계산?

확정 계산 순서:
  1단계 (가산): 기본ATK + 장비 + 스펠버프 + 킬보너스 + 특성 = 합산ATK
  2단계 (승산): 합산ATK × (1 + RAGE%) × 원소배율 = 최종ATK
  3단계 (크리): 최종ATK × 크리티컬배율(×2) = 최종데미지

  예시:
    기본 ATK 4 + 장비 +7 + 킬3회(+6) + 특성(+3) = 20
    RAGE 30 (30%) → 20 × 1.3 = 26
    원소 유리 (×1.5) → 26 × 1.5 = 39
    크리티컬 (×2) → 39 × 2 = 78 데미지!

  핵심: 가산(+) 먼저 → 승산(×) 나중에 → 크리 마지막
```

## 8. 기타 엣지케이스

### 동시 사망
```
두 유닛이 같은 턴에 서로 죽이면?
→ SPD 순서대로 처리, 먼저 행동한 유닛의 공격이 먼저 적용
→ 먼저 죽은 유닛은 반격 불가
```

### 부활 + 사령술 충돌
```
부활 키워드 있는 유닛이 사령술로 죽으면?
→ 부활이 우선 (부활로 살아남으면 사령술 발동 안 함)
→ 부활 이미 사용했으면 → 사령술 발동
```

### 빙결 + 분노 100
```
빙결 상태에서 RAGE 100 도달하면?
→ 빙결 해제 후 다음 턴에 궁극기 발동 (빙결 중에는 행동 불가)
→ RAGE는 100 유지 (빙결 중 리셋 안 함)
```

### 도발 유닛 사망
```
도발 유닛이 턴 중간에 사망하면?
→ 도발 즉시 해제, 나머지 공격자는 다른 타겟 선택
```

### 전투 무한 루프 방지
```
양쪽 다 힐만 하고 공격 안 하면?
→ 라운드 30 도달 시 강제 종료 → 남은 HP 비율로 승패 결정
→ HP 비율 동일하면 → 무승부 (양쪽 소량 보상)
```
