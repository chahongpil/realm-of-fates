## 2. 보석↔골드 환율

```
💎1 = 💰2

예시:
  💎50 = 💰100
  💎100 = 💰200
```

### 구매별 화폐 사용 📦 (가챠 행 DEPRECATED)

> 🚫 **"영웅 가챠 단일/10연차" 2 행 + "유닛·스킬·장비 뽑기" 4 행은 v1 잔재.**
> 유닛·스킬·장비는 v2 에서 **6 암시장 골드 구매** (`monetization.md` § 8). 영웅 가챠는 **폐기** (신 카드는 확정 지급, `monetization.md` § 3).
> 나머지 행 (선술집 새로고침 · 건물 건설 · 단련 · 교회 치료 · 체력 회복 · 골드 교환) 은 유효.

| 컨텐츠 | 💰 골드 | 💎 보석 | 보석 이득 |
|--------|--------|--------|----------|
| 유닛 단일 뽑기 | 5💰 | 💎2 (=4💰) | 💰1 이득 |
| 유닛 10연차 | 45💰 | 💎20 (=40💰) | 💰5 이득 |
| 스킬 단일 뽑기 | 3💰 | 💎1 (=2💰) | 💰1 이득 |
| 장비 단일 뽑기 | 8💰 | 💎3 (=6💰) | 💰2 이득 |
| **영웅 가챠 단일** | — | 💎30 | 보석 전용 |
| **영웅 가챠 10연차** | — | 💎250 (50 할인) | 보석 전용 |
| 선술집 새로고침 | 1💰 | — | 골드만 |
| 건물 건설 | 💰만 | — | 골드만 |
| 단련 | 💰만 | — | 골드만 |
| 교회 치료 | 💰만 | — | 골드만 |
| 체력 회복 (추가 전투) | — | 💎3 | 보석만 |
| 골드 교환 | — | 💎10→💰25 | 보너스 +5 |

> 보석으로 가챠하면 골드보다 약간 이득 → 과금 유인

## 3. 운명의 실타래 + 신의 잔재 (v2 2026-04-22)

> 구 "영웅 가챠" 섹션은 **삭제됨** (`design/monetization.md` v2.1 § 0 "가챠 없음 확정"). 신 카드 획득은 **확정 지급** 3 경로 (싱글 엔딩 조각 누적 / PvP 랭크 + 원소 선택 / 시즌 이벤트 조각 누적) — `monetization.md` § 3 참조.

### 3.1 운명의 실타래 (Thread of Fate)

**PvP 경제의 근간**. 2 종 분리로 P2W 방지.

| 종류 | 획득 | 용도 | 시즌 1 위 집계 |
|---|---|---|---|
| 🧵 **여신의 실 가닥** (Strand) | PvP 승리 시 상대에게서 흡수 | 시즌 랭크 포인트 | ✅ |
| 🪡 **기워낸 실 조각** (Forged Thread) | 6 암시장 골드 구매 | PvP 매칭 참가비 (입장권) | ❌ |

로어 근거: 신들도 여신의 진짜 실엔 접근 못 함 → 6 신이 원소력으로 **모조 실** 을 빚음. 옥체는 원소 각인을 알아보고 **여신의 실만** 신좌 자격으로 인정. 모조 실은 "참가권" 까지만 (`lore-bible.md` v2 § 10).

### 3.2 신의 잔재 (🔮 Divine Remnant)

**캐릭터 신 카드 소멸 시** 획득 (`monetization.md` § 9).

- **획득**: 신 카드 2 시즌 수명 종료 시 소멸 → 잔재 획득 (기본 50~100)
- **용도**: 전설 카드 코팅 업그레이드 · 특수 스킨 · 카드 레벨업 XP 가속
- **제약**:
  - 🚫 잔재로 신 카드 재구매 **불가** (수명의 감정선 보호)
  - 🚫 보석으로 잔재 구매 **불가** (무과금 전용 경로)
  - 🚫 P2W 방지: 순위에 영향 X

### 3.3 경제 순환

```
신 카드 획득 (싱글 엔딩 · PvP 랭크 · 이벤트 확정 지급)
      ↓ 2 시즌 사용
  수명 다함 → 소멸 의식 (꺼진 신좌의 방)
      ↓
 🔮 신의 잔재 획득
      ↓ 사용
전설 유물 Lv업 / 코팅 업그레이드 / 특수 강화
      ↓ 전투력 유지
 다음 시즌 도전
```

## 4. 축복 ✨ (특수 화폐)

### 획득 방법
```
3연승: ✨1
5연승: ✨2 (추가)
10연승: ✨3 (추가)
업적 달성: ✨1~3 (업적별)
이벤트 보상: ✨1~5
신의영역 리그 시즌 보상: ✨10 (시즌당 1회)
```

### 사용처 📦 (가챠 항목 DEPRECATED)

> 🚫 **"전설 뽑기권" · "영웅 가챠 티켓" 2 행과 맨 아래 안내문은 v1 잔재.**
> 나머지 (랜덤 골드 장비 · 이름 변경권 · 유닛 부활권 · 전직 초기화권 · 코팅 랜덤 박스) 는 유효.

| 상품 | 가격 |
|------|------|
| 전설 뽑기권 (일반 가챠) | ✨3 |
| 랜덤 골드 장비 | ✨5 |
| 이름 변경권 | ✨2 |
| 유닛 부활권 (장례 복구) | ✨3 |
| 전직 초기화권 | ✨10 |
| 코팅 랜덤 박스 | ✨5 |
| **영웅 가챠 티켓** (✨→💎 교환) | ✨20 → 💎30 상당 |

> 영웅 가챠는 기본적으로 💎지만, 무과금 유저는 ✨ 축적 후 교환 가능.

## 4-A. 연패 안전망 + 골드 이자 (2026-04-12 추가, TFT 레퍼런스)

📚 레퍼런스: `.claude/references/card-game-studies/tft_autobattler_core.md`

### 연패 보상 스트릭
패배해도 포기하지 않도록, 연패할수록 추가 보상.

| 연패 횟수 | 기본 5g에 추가 |
|----------|---------------|
| 2연패 | +1g |
| 3연패 | +3g + 브론즈 유닛 무료 1체 |
| 4연패 | +4g |
| 5연패 | +5g + 실버 유닛 무료 1체 |
| 6연패+ | +5g 고정 |

**목적**: 바닥 치는 유저의 회복 가능성 유지. 포기 방지.
**승리 시**: 연패 스트릭 초기화.

### 골드 이자 (선택적 도입)
저축 인센티브로 "지금 쓸까 모을까" 긴장감 유도.

| 보유 골드 | 이자 (매 전투 종료 후) |
|----------|----------------------|
| 10g 이상 | +1g |
| 20g 이상 | +2g |
| 30g 이상 | +3g |
| 50g 이상 | +5g (최대) |

**주의**: 도입 시 현재 경제 밸런스(5-1) 재검증 필수. 인플레이션 가능성 있음.

## 5. 화폐 획득 요약 (일일 기준) 📦 DEPRECATED

> 🚫 **섹션 전체 폐기 (2026-04-22 가챠 없음 확정).**
> 아래 "무과금 하루 N 전투 시 💎 Y → 영웅 가챠 Z 회" 같은 계산은 v1 가챠 모델 기반. v2 는 **확정 지급 경로** (싱글 엔딩 / PvP 랭크 / 시즌 이벤트) 라 시뮬 의미 없음.
> v2 경제 시뮬은 별도 문서로 관리 예정 (미작성). 현재 v2 경제 지침은 `design/monetization.md` § 8 골드 경제 + § 9 신의 잔재 참조.
> 이 섹션은 **역사 기록용으로만 남겨둔다**. 수치 인용 금지.


### 무과금 유저 (하루 8전투 기준)
```
💰 골드: ~125/일
  전투 승리 5회 × 12 = 60
  전투 패배 3회 × 5 = 15
  루트박스 평균: 45
  퀘스트: ~20

💎 보석: ~5/일
  업적/이벤트/시즌패스 무료트랙

✨ 축복: ~2/일
  연승 보너스
```

### 라이트 과금 ($5/월)
```
💰 골드: ~150/일 (시즌패스 보너스)
💎 보석: ~15/일 (시즌패스)
✨ 축복: ~3/일
```

### 경제 밸런스 체크
```
무과금 3개월 기준:
  💰 ~11,250 (건물 대부분 건설 가능)
  💎 ~450 (일반 10연차 22회 or 영웅 가챠 15연차)
  ✨ ~180 (전직 초기화 18회 or 영웅 티켓 9매 = 💎270 상당)

핵심:
  일반 유닛/장비/스펠 = 무과금으로 충분히 모음
  영웅 가챠 = 무과금은 느리지만 가능 (월 5~8 뽑기)
  과금은 "빠르게" 도달, 최종 도달점은 동일 (천장 존재)
```
