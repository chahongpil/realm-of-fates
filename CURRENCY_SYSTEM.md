# 화폐 시스템 상세

> 마지막 업데이트: 2026-04-12 (4종 → 3종 통합, 👑신의은총 폐지)

---

## 1. 화폐 3종

| 등급 | 화폐 | 아이콘 | 획득 방법 | 주 용도 |
|------|------|--------|----------|--------|
| 기본 | **골드** | 💰 | 전투/퀘스트/상점 | 건설, 제작, 단련, 일반 가챠 |
| 프리미엄 | **보석** | 💎 | 과금/시즌패스/업적 | 일반 가챠, **영웅 가챠**, 편의, 골드 교환 |
| 특수 | **축복** | ✨ | 연승/업적/이벤트 | 특별 상점, 전직 초기화, 코팅 박스 |

> ⚠️ 이전 기획의 **👑 신의은총** 화폐는 **폐지**. 영웅 가챠는 💎 보석으로 흡수됨.

---

## 2. 보석↔골드 환율

```
💎1 = 💰2

예시:
  💎50 = 💰100
  💎100 = 💰200
```

### 구매별 화폐 사용

| 컨텐츠 | 💰 골드 | 💎 보석 | 보석 이득 |
|--------|--------|--------|----------|
| 유닛 단일 뽑기 | 5💰 | 💎2 (=4💰) | 💰1 이득 |
| 유닛 10연차 | 45💰 | 💎20 (=40💰) | 💰5 이득 |
| 스킬 단일 뽑기 | 3💰 | 💎1 (=2💰) | 💰1 이득 |
| 장비 단일 뽑기 | 8💰 | 💎3 (=6💰) | 💰2 이득 |
| **영웅 가챠 단일** | — | 💎30 | 보석 전용 |
| **영웅 가챠 10연차** | — | 💎250 (50 할인) | 보석 전용 |
| 선술집 새로고침 | 1💰 | — | 골드만 |
| 건물 건설 | 💰만 | — | 골드만 |
| 단련 | 💰만 | — | 골드만 |
| 교회 치료 | 💰만 | — | 골드만 |
| 체력 회복 (추가 전투) | — | 💎3 | 보석만 |
| 골드 교환 | — | 💎10→💰25 | 보너스 +5 |

> 보석으로 가챠하면 골드보다 약간 이득 → 과금 유인

## 3. 영웅 가챠 (💎 보석 전용)

### 뽑기 가격
```
단일 뽑기: 💎30
10연차: 💎250 (50 보석 할인)
```

### 영웅 가챠 확률
```
브론즈 영웅: 40%
실버 영웅:  30%
골드 영웅:  20%
전설 영웅:   8%
신 영웅:     2%
```

### 천장 시스템
```
10연차 시 실버 이상 확정
30연차 시 골드 이상 확정
50연차 시 전설 확정 (💎1,250)
100연차 시 신 영웅 확정 (💎2,500)
```

### 영웅 가챠 vs 일반 가챠 차이

| | 일반 가챠 (선술집) | 영웅 가챠 |
|---|---|---|
| 화폐 | 💰 골드 / 💎 보석 | 💎 **보석 전용** (고비용) |
| 뽑히는 것 | 일반 유닛 (소모품) | **영웅 카드** (탱커, 플레이어 본체) |
| 중복 시 | 유닛 추가 | 명예 +50 변환 |
| 가격대 | 💰5 / 💎2 | 💎30 (15배 고가) |
| 의미 | 동료 모집 | **새 영웅 = 새 플레이 스타일** |

## 4. 축복 ✨ (특수 화폐)

### 획득 방법
```
3연승: ✨1
5연승: ✨2 (추가)
10연승: ✨3 (추가)
업적 달성: ✨1~3 (업적별)
이벤트 보상: ✨1~5
신의영역 리그 시즌 보상: ✨10 (시즌당 1회)
```

### 사용처
| 상품 | 가격 |
|------|------|
| 전설 뽑기권 (일반 가챠) | ✨3 |
| 랜덤 골드 장비 | ✨5 |
| 이름 변경권 | ✨2 |
| 유닛 부활권 (장례 복구) | ✨3 |
| 전직 초기화권 | ✨10 |
| 코팅 랜덤 박스 | ✨5 |
| **영웅 가챠 티켓** (✨→💎 교환) | ✨20 → 💎30 상당 |

> 영웅 가챠는 기본적으로 💎지만, 무과금 유저는 ✨ 축적 후 교환 가능.

## 4-A. 연패 안전망 + 골드 이자 (2026-04-12 추가, TFT 레퍼런스)

📚 레퍼런스: `.claude/references/card-game-studies/tft_autobattler_core.md`

### 연패 보상 스트릭
패배해도 포기하지 않도록, 연패할수록 추가 보상.

| 연패 횟수 | 기본 5g에 추가 |
|----------|---------------|
| 2연패 | +1g |
| 3연패 | +3g + 브론즈 유닛 무료 1체 |
| 4연패 | +4g |
| 5연패 | +5g + 실버 유닛 무료 1체 |
| 6연패+ | +5g 고정 |

**목적**: 바닥 치는 유저의 회복 가능성 유지. 포기 방지.
**승리 시**: 연패 스트릭 초기화.

### 골드 이자 (선택적 도입)
저축 인센티브로 "지금 쓸까 모을까" 긴장감 유도.

| 보유 골드 | 이자 (매 전투 종료 후) |
|----------|----------------------|
| 10g 이상 | +1g |
| 20g 이상 | +2g |
| 30g 이상 | +3g |
| 50g 이상 | +5g (최대) |

**주의**: 도입 시 현재 경제 밸런스(5-1) 재검증 필수. 인플레이션 가능성 있음.

## 5. 화폐 획득 요약 (일일 기준)

### 무과금 유저 (하루 8전투 기준)
```
💰 골드: ~125/일
  전투 승리 5회 × 12 = 60
  전투 패배 3회 × 5 = 15
  루트박스 평균: 45
  퀘스트: ~20

💎 보석: ~5/일
  업적/이벤트/시즌패스 무료트랙

✨ 축복: ~2/일
  연승 보너스
```

### 라이트 과금 ($5/월)
```
💰 골드: ~150/일 (시즌패스 보너스)
💎 보석: ~15/일 (시즌패스)
✨ 축복: ~3/일
```

### 경제 밸런스 체크
```
무과금 3개월 기준:
  💰 ~11,250 (건물 대부분 건설 가능)
  💎 ~450 (일반 10연차 22회 or 영웅 가챠 15연차)
  ✨ ~180 (전직 초기화 18회 or 영웅 티켓 9매 = 💎270 상당)

핵심:
  일반 유닛/장비/스펠 = 무과금으로 충분히 모음
  영웅 가챠 = 무과금은 느리지만 가능 (월 5~8 뽑기)
  과금은 "빠르게" 도달, 최종 도달점은 동일 (천장 존재)
```
