## 1. 카드 일러스트 (SD + LoRA 생성)

### 유닛 카드 (현재 42종 → 최종 300+종)

| 카테고리 | 수량 | 스타일 | 생성 방법 |
|---------|------|--------|----------|
| 영웅 카드 (6원소×3역할) | 18종 | 고퀄리티, 상세 | LoRA + 후보정 |
| 브론즈 유닛 | 10종 → 50종 | 간결, 실루엣 강조 | LoRA 배치 |
| 실버 유닛 | 7종 → 40종 | 중간 디테일 | LoRA 배치 |
| 골드 유닛 | 5종 → 30종 | 상세, 이펙트 포함 | LoRA + 후보정 |
| 전설 유닛 | 3종 → 15종 | 매우 상세, 화려 | LoRA + 외주 보정 |
| 신 유닛 | 1종 → 5종 | 최고 퀄리티 | 개별 제작 |

### LoRA 트리거 + 프롬프트 가이드
```
트리거: rofcard
스타일: "fantasy card game illustration, stylized digital painting, 
        dramatic cinematic lighting, dark atmospheric, portrait composition"
금지어: hearthstone, blizzard, warcraft, wow
```

## 2. 스킬/스펠 아이콘 (100+종)

### 아이콘 스타일
```
크기: 128×128px (게임 내), 256×256 (원본)
형태: 원형 or 사각 라운드 (프레임 없음)
배경: 원소 색상 그라데이션
요소: 중앙에 심볼, 글로우 효과
```

### 아이콘 카테고리

| 카테고리 | 수량 | 예시 |
|---------|------|------|
| 공격 스킬 | 15종 | 검격, 화살, 번개 등 |
| 방어 스킬 | 10종 | 방패, 도발, 철갑 등 |
| 지원 스킬 | 12종 | 치유, 버프, 부활 등 |
| 디버프 | 8종 | 빙결, 독, 화상, 저주 등 |
| 유물 | 12종 | 반지, 검, 방패, 오브 등 |
| 장비 (무기) | 15종 | 검, 활, 지팡이, 도끼 등 |
| 장비 (방어구) | 10종 | 갑옷, 로브, 가죽 등 |
| 장비 (장신구) | 10종 | 반지, 목걸이, 부적 등 |
| UI 아이콘 | 20종 | 골드, 보석, 축복, 설정 등 |

### SD 아이콘 생성 프롬프트
```
rofcard icon, [설명], game icon style, clean design, 
centered composition, dark gradient background, 
glowing effect, 128x128, simple shapes, no text
```

## 4. 전투 이펙트/파티클 (30+종)

### VFX 카테고리

| 카테고리 | 수량 | 설명 | 생성 방법 |
|---------|------|------|----------|
| **타격 이펙트** | 6종 | 물리/마법/원소별 | Canvas 2D + CSS keyframes |
| **스킬 발동** | 12종 | 키워드별 이펙트 | CSS keyframes + PNG sprites |
| **버프/디버프** | 8종 | 오라, 아이콘 위 빛남 | CSS filter/box-shadow + Particle JS |
| **사망** | 3종 | 일반/영웅/보스 | CSS transform + opacity |
| **소환** | 2종 | 유닛 등장, 부활 | CSS scale + fade-in |
| **치유** | 3종 | 단일/전체/대치유 | Canvas 2D particles |

## 5. UI 프레임/배경 (카드 UI 새 디자인)

### 카드 프레임 5등급
| 등급 | 프레임 색상 | 질감 | 특수 효과 |
|------|-----------|------|----------|
| 브론즈 | 어두운 동색 | 거친 석재 | 없음 |
| 실버 | 차가운 은색 | 매끈한 금속 | 미세 광택 |

| 전설 | 보라색 | 마법 룬 | 파티클 + 글로우 |
| 신 | 빨간색 + 금색 | 고대 문양 | 맥동 + 오라 |

### UI 패널
| 패널 | 용도 | 스타일 |
|------|------|--------|
| 메인 패널 | 팝업, 모달 | 다크 석재 + 미세 테두리 |
| HUD 바 | 상단 정보 | 반투명 다크 + 블러 |
| 버튼 | 일반/확인/취소 | 플랫 + 약간 입체 |
| 툴팁 | 키워드 설명 | 등급별 테두리 + 원소 배경 |

## 9. 에셋 생성 도구 체인 (웹 기반)

```
카드 일러스트: SD 1.5 + DreamShaper → GIMP 후보정 → img/ 폴더 PNG
스킬 아이콘: SD 1.5 (icon mode) → 128×128 PNG → img/skill_*.png
VFX: Canvas 2D 파티클 + CSS keyframes + box-shadow glow
2D 건물: SD 생성 PNG (levelup 단계별) → img/building_*_lv{1-5}.png
NPC: SD 생성 2D 스프라이트 → img/npc_*.png
UI: Figma 디자인 → HTML/CSS 직접 구현 (index.html)
사운드: Suno AI (BGM) + Freesound.org (SFX) → snd/ + Web Audio API
애니메이션: CSS @keyframes + requestAnimationFrame + GSAP (필요 시)
```
