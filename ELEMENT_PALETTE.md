> 6원소 고유 색감 확정 — 카드 프레임 엣지 그라데이션, 파티클, UI 강조 색상 통일

## 철학

- 각 원소는 **2색 그라데이션** (어두운 메인 → 밝은 하이라이트)
- 카드 프레임 엣지에 이 그라데이션을 은은하게 적용 → 한눈에 원소 구분
- 파티클/이펙트/UI 강조 색상도 이 팔레트에서 파생

## 적용 영역

### 1. 카드 프레임 엣지 그라데이션
```css
/* 예: 불 원소 카드 */
.card[data-element="fire"] {
  border-image: linear-gradient(135deg, #7a1f0a, #ff8c3a) 1;
  box-shadow: 0 0 12px rgba(255, 140, 58, 0.3);
}
```

### 2. 원소 태그 (카드 이름 아래)
- 배경: 어두운 메인 (`#7a1f0a` 등)
- 텍스트: 밝은 하이라이트 (`#ff8c3a`)
- 아이콘: 이모지 그대로

### 3. 파티클 이펙트
- 기본 색: 밝은 하이라이트
- 잔광: 어두운 메인으로 페이드
- 크리티컬: 순간 흰색(`#fff`) 플래시 후 원소 색 복귀

### 4. UI 강조 (원소 필터, 시너지 활성)
- 원소 선택 화면: 각 원소 카드 테두리에 해당 그라데이션
- 시너지 바: 활성 시 원소 하이라이트 색 글로우

## 배색 금지 사항
- ❌ 두 원소 색상을 **혼합** 사용 금지 (복합 원소 개념 없음)
- ❌ 하이라이트 색을 배경에 크게 사용 금지 (피로도 상승)
- ❌ RGB 순정(#ff0000 등) 금지 — 모두 우리 팔레트 내에서만

## 접근성
- 색맹 유저를 위해 **아이콘은 항상 동반**
- 색상만으로 원소 구분하지 말 것 — 텍스트/이모지 병기 필수
