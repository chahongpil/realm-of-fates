## 5. 장비 강화

```
강화 = 같은 슬롯 장비를 재료로 사용하여 스탯 업

비용: 재료 장비 1개 + 골드
  브론즈 강화: 5💰 / 주스탯 +1
  실버 강화: 15💰 / 주스탯 +2
  골드 강화: 40💰 / 주스탯 +3
  전설 강화: 100💰 / 주스탯 +5

최대 강화: +10 (강화 10회)
표시: 강철검 +3, 가시갑옷 +7 등

강화 실패 없음 (100% 성공)
```

## 6. 장비 분해

```
장비를 분해하면 재료로 변환

분해 수율 (대장간 레벨에 따라 증가):
  Lv1: 제작 재료의 50% 회수
  Lv2: 60% 회수
  Lv3: 70% 회수 + 마정석 조각 1개 보너스
  Lv4: 80% 회수
  Lv5: 90% 회수 + 마정석 덩어리 1개 보너스

예시: 실버 무기 분해 (Lv3 대장간)
  원래 재료: 덩어리 2 + 조각 2
  회수: 덩어리 1 + 조각 1 + 마정석조각 1 (70%)
```

## 7. 장비 획득 경로 요약

| 획득 방법 | 내용 | 확률/조건 |
|----------|------|----------|
| **전투 드랍** | 재료 드랍 (조각~결정) | 승리 시 40~10% |
| **대장간 제작** | 재료 + 골드 → 장비 | 확정 |
| **상점 구매** | 재료 직접 구매 | 상점 Lv3+ |
| **퀘스트 보상** | 재료 또는 완성 장비 | 퀘스트별 |
| **레이드 보상** | 전설~신 재료 | 레이드 전용 |
| **장비 분해** | 기존 장비 → 재료 | 확정 |
| **가챠 (대장간)** | 랜덤 장비 직접 뽑기 | 8💰/회 |
