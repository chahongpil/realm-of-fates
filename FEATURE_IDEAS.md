## 1. 즉시 구현 가능 (Phase 1.5)

### 1-1. 카드 키워드 시스템 (업계 표준 툴팁 패턴)
**현재**: 스킬 설명이 길고 형식이 제각각
**개선**: 공통 키워드를 정의하고, 호버/클릭 시 설명 표시

| 키워드 | 아이콘 | 효과 | 해당 유닛 |
|--------|--------|------|-----------|
| **도발** | 🛡️ | 적이 이 유닛을 먼저 공격해야 함 | 수비병, 기사, 성기사 |
| **선제** | ⚡ | 항상 먼저 공격 | 사냥꾼, 창병, 썬더버드 |
| **관통** | 🎯 | 방어력 무시 | 석궁병, 저격수 |
| **급소** | 🗡️ | N% 확률로 3배 피해 | 도적, 암살자 |
| **광기** | 🔥 | HP 50% 이하 시 공격력 2배 | 광전사 |
| **흡혈** | 💜 | 피해의 N% HP 회복 | 죽음의기사, 리치 |
| **부활** | ✨ | 첫 사망 시 HP 50%로 부활 | 불사조 |
| **광역** | 💥 | 모든 적에게 피해 | 화염술사, 드래곤, 견습마법사 |
| **빙결** | ❄️ | 1턴 행동 불가 | 빙결술사 |
| **화상** | 🔥 | 턴마다 지속 피해 | 불꽃정령, 화염술사 |
| **무리** | 🐺 | 같은 종족 1체당 공격+N | 늑대 |
| **은신** | 👤 | 다음 턴 회피 100% | 도적, 암살자 (처치 시) |

**구현**: 각 유닛의 skillDesc를 키워드 기반으로 단축, `<b>` 태그로 표시, 
키워드 클릭 시 툴팁 팝업

### 1-2. 카드 플레이버 텍스트
업계 표준 카드게임 매력 요소. 각 카드에 세계관 몰입형 한줄 문구 추가.

**예시:**
| 유닛 | 플레이버 텍스트 |
|------|----------------|
| 민병 | *"쟁기를 내려놓고 칼을 들었다. 이유는 묻지 마라."* |
| 늑대 | *"한 마리는 사냥꾼이지만, 무리는 재앙이다."* |
| 수비병 | *"이 문은 내가 서 있는 한 열리지 않는다."* |
| 도적 | *"등 뒤의 칼날은 보이지 않는 법이다."* |
| 기사 | *"맹세는 한 번이면 충분하다. 평생을 지키면 되니까."* |
| 암살자 | *"이름을 모르는 자가 가장 위험한 자다."* |
| 광전사 | *"피가 끓을수록 칼날이 날카로워진다."* |
| 드래곤 | *"산맥을 넘는 그림자. 그것이 마지막으로 보는 것이다."* |
| 대천사 | *"최후의 심판은 자비로 시작되고, 빛으로 끝난다."* |
| 리치 | *"죽음은 끝이 아니다. 나에게는 시작이다."* |
| 타이탄 | *"번개가 치면 세계가 떨린다. 그가 걸으면 대지가 갈라진다."* |

### 1-3. 전투 중 시너지 표시
현재 종족/원소 시너지가 코드에는 있지만 UI에 표시되지 않음.

**구현 아이디어:**
- 전투 화면 상단에 활성 시너지 아이콘 표시
- 예: "🐺 야수 x2 → 공격+2" 같은 배지
- 유닛 배치할 때 어떤 시너지가 발동하는지 미리보기
- 업계 표준 "메카닉 하이라이트"처럼 시너지 발동 시 번쩍임

---

### 1-4. 전투 로그 (좌측 히스토리 바)
좌측에 최근 행동을 시각적으로 표시하는 히스토리 바 패턴.

```
전투 로그 (최근 5개):
🗡️ 광전사 → 적 민병 12 데미지 (급소!)
❄️ 빙결술사 → 적 기사 빙결 1턴
💀 적 민병 처치됨
🩸 죽음의기사 흡혈 → HP +6
✨ 사제 대치유 → 아군 전체 HP +4
```

전투가 자동으로 진행되기 때문에 무슨 일이 일어났는지 알기 어려움 → 로그가 필수.

## 2. 중기 구현 (Phase 2)

### 2-1. 일일 퀘스트 시스템
업계 표준 일일 퀘스트 패턴. 매일 목표를 주고 보상 제공.

| 퀘스트 | 조건 | 보상 |
|--------|------|------|
| 전사의 날 | 전사 유닛 3체 이상으로 승리 | 💰 5 |
| 불의 시련 | 화염 원소 유닛만으로 전투 | 💰 8 |
| 약자의 반란 | 브론즈 유닛만으로 승리 | ✨ 실버 스킬 1개 |
| 무패행진 | 3연승 달성 | 💰 10 |
| 학살자 | 한 전투에서 10체 이상 처치 | 🎲 랜덤 유물 |
| 수호자 | 영웅 HP 50% 이상으로 승리 | 💰 5 |
| 탐험가 | 7라운드 이상 생존 | ✨ 골드 스킬 1개 |

- 매일 3개 랜덤 배정 (리셋: 자정)
- 완료하지 못한 퀘스트는 다음 날 1개만 유지

---

### 2-2. 업적(도전과제) 시스템

| 업적 | 조건 | 보상 | 칭호 |
|------|------|------|------|
| 첫 승리 | 리그 첫 승리 | 💰 10 | "신의 첫걸음" |
| 10연승 | 10연속 승리 | 전설 유물 1개 | "불패의 전략가" |
| 수집광 | 유닛 20종 보유 | 💰 50 | "만물의 신" |
| 드래곤 슬레이어 | 드래곤을 10회 처치 | 특수 스킬 | "용살자" |
| 재기의 신 | HP 10% 이하에서 역전승 | 특수 유물 | "불사의 의지" |
| 부자 신 | 골드 1000 달성 | 신급 유닛 1개 | "황금신" |
| 최고의 신 | 신의영역 리그 달성 | 한정 일러스트 | "절대신" |

업적 달성 시 화면 중앙에 황금 배너 + 팡파레 연출.

---

### 2-3. 카드 진화 시스템 (전직)
GDD에 있는 전직 시스템을 구체화.

```
기사 (Lv10, 명예50)
  ├── 성기사 (holy 강화) — 신성 치유 + 도발 강화
  └── 암흑기사 (dark 전환) — 흡혈 + 공격 강화

민병 (Lv10, 명예30)  
  ├── 근위병 — 도발 획득 + 방어 대폭 상승
  └── 검투사 — 공격 대폭 상승 + 광기 획득

견습마법사 (Lv10, 명예30)
  ├── 화염술사 (fire) — AOE 강화
  └── 빙결술사 (water) — CC(군중제어) 강화
```

전직 시 카드 일러스트가 변경되고, 화려한 진화 연출 (메탈릭 스킨 전환 느낌).

---

### 2-4. 카드 코팅/스킨 시스템
업계 표준 메탈릭 변형 스킨 시스템. 상세는 `PHASE3_COATING_PLAN.md` 참조.

| 코팅 | 획득 방법 | 효과 |
|------|----------|------|
| 일반 | 기본 | 기본 일러스트 |
| 빛나는 (홀로그램) | 10연승 | 카드에 홀로그램 반짝임 |
| 황금 | 리그 보상 | 황금 테두리 + 애니메이션 일러스트 |
| 신화 | 신의영역 달성 | 특수 이펙트 + 전용 BGM |

순전히 시각적 보상으로, 수집 욕구를 자극.

## 3. 장기 구현 (Phase 3~4)

### 3-1. 원소 상성 시스템
전투에 전략적 깊이 추가.

```
🔥 화염 → ❄️ 빙결 (1.5배)
❄️ 빙결 → ⚡ 번개 (1.5배)  
⚡ 번개 → 🌍 대지 (1.5배)
🌍 대지 → 🔥 화염 (1.5배)
☀️ 신성 ↔ 🌑 암흑 (서로 1.5배)
```

상성 유리 시 데미지 1.5배, 불리 시 0.75배.
카드 선택 시 상대 영웅의 원소가 표시되어 전략적 편성 유도.

---

### 3-2. 길드(결사) 시스템
회원가입 시 입력하는 "결사"를 실제 길드로 활용.

- 같은 결사 이름의 유저끼리 자동 그룹핑
- 길드 레이드 보스 (협동)
- 길드 대전 (주간 토너먼트)
- 길드 건물 (공유 보너스)

---

### 3-3. 이벤트 전투 모드

| 이벤트 | 규칙 | 보상 |
|--------|------|------|
| 야수의 밤 | 야수 유닛만 사용 가능 | 한정 야수 유닛 |
| 거인의 행진 | 모든 유닛 스탯 2배 | 한정 유물 |
| 역전의 신 | HP 역전 (높을수록 불리) | 특수 스킬 |
| 1대1 결투 | 영웅만으로 전투 | 칭호 |
| 무한 던전 | 끝없이 이어지는 라운드 | 라운드별 누적 보상 |

## 4. UX 개선 아이디어

### 4-1. 튜토리얼 개선
- 현재 텍스트 기반 → **인터랙티브 튜토리얼**로
- 첫 전투를 "연습 전투"로 만들어 패배해도 불이익 없도록
- 각 건물 첫 방문 시 NPC가 사용법 설명 (말풍선)

### 4-2. 전투 속도 조절
- 1x / 2x / 3x 속도 버튼
- 업계 표준 "빨리감기" 느낌

### 4-3. 마을 건물 업그레이드 시각 변화
- 건물 레벨에 따라 외형 변화 (이미 이미지는 만들어둠)
- 업그레이드 시 건설 애니메이션
- NPC 초상화도 레벨에 따라 변경 (이미 있음)

### 4-4. 카드 상세 화면 개선
- 카드를 길게 누르면 풀스크린 확대 (업계 표준 제스처)
- 3D 회전 효과 (CSS perspective)
- 스탯 비교 (평균 대비 높으면 녹색, 낮으면 빨간색)
