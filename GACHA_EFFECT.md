## 3. 등급별 등장 이펙트

### 브론즈
```
이펙트: 없음, 조용히 공개
사운드: 가벼운 카드 넘기는 소리 "슥"
카메라: 변화 없음
```

### 실버
```
이펙트: 은색 반짝임 (카드 테두리 1회 빛남)
사운드: 살짝 반짝 소리 "딩"
카메라: 변화 없음
```

### 골드 ★
```
이펙트:
  - 금빛 폭발 파티클 (카드 중심에서 방사)
  - 화면 전체 살짝 밝아짐 (0.3초)
  - 카드 테두리 금색 glow 지속

사운드: 짧은 팡파레 "빠빠빰!"
카메라: 살짝 줌인 (1.1배)
```

### 전설 ★★
```
이펙트:
  - 보라색 번개가 카드를 감쌈
  - 카드가 화면 중앙으로 확대 (1.5배)
  - 화면 흔들림 (0.5초)
  - 보라색 오라 파티클 지속
  - 배경 완전 어두워짐 + 스포트라이트

사운드: 웅장한 팡파레 "빠~바바바밤!!"
카메라: 줌인 (1.5배) + 슬로모션 (0.3초)
```

### 신 ★★★
```
이펙트:
  - 화면 깨짐 이펙트 (유리 깨지는 느낌)
  - 빨간 + 금색 폭발 (화면 전체)
  - 카드가 공중으로 떠오르며 회전 (360도)
  - 슬로모션 1초
  - 번개 + 불꽃 + 빛줄기 파티클 동시
  - 카드 뒤에 원소 신의 인장 나타남

사운드:
  0.0초: 뇌성벽력 "쿠과과과광!"
  0.5초: 천상의 합창 "아아아아~"
  1.5초: 심장박동 소리

카메라:
  줌인 (2배) → 슬로모션 → 카드 확대 → 원래로
```

## 5. 영웅 가챠 특별 연출 (💎 보석 전용)

### 일반 가챠와 차이
```
배경: 일반 = 어두운 선술집
      영웅 = 신전 + 신의 빛줄기

카드 뒷면: 일반 = 석재 질감
           영웅 = 금빛 + 신의 인장 + 오라

뒤집기: 일반 = Y축 회전
        영웅 = 빛 기둥 속에서 카드 하강 + 회전

등장: 영웅 등급 무관하게 항상 특별 이펙트
      (영웅 자체가 귀하니까)
```

## 7. 전설/신 카드 움직이는 일러스트 (셰이더 애니메이션)

> 전설 이상 카드는 일러스트가 살아 움직임 (메탈릭 변형 스킨 애니메이션 방식)
> 정지 이미지 1장 + CSS filter/mask + Canvas 오버레이로 영역별 애니메이션

### 적용 대상
| 등급 | 움직임 |
|------|--------|
| 브론즈 | 정지 |
| 실버 | 정지 |
| 골드 | 정지 |
| **전설** | **움직임 (약하게)** |
| **신** | **움직임 (강하게 + 파티클)** |

### 전설 카드 움직임 효과
```
강도: 약하게 (은은한 생동감)
  - 눈/보석: 발광 맥동 (1초 주기)
  - 망토/머리카락: 바람에 살짝 흔들림
  - 배경: 느리게 스크롤 (구름/연기)
  - 무기: 미세한 빛 반짝임
```

### 신 카드 움직임 효과
```
강도: 강하게 (압도적 존재감)
  - 전체 배경: 움직임 (폭풍/화염/파도)
  - 캐릭터 오라: 에너지 흐름 (원소 색상)
  - 눈: 강한 발광 맥동 (0.5초 주기)
  - 원소 이펙트:
    🔥 불: 화염 흔들림 + 불씨 상승
    💧 물: 물결 일렁임 + 얼음 반짝
    ⚡ 번개: 전기 아크 깜빡 + 스파크
    🌿 대지: 잎사귀 흩날림 + 덩굴 성장
    🌑 암흑: 보라 안개 흐름 + 해골 페이드
    ✨ 신성: 빛줄기 하강 + 깃털 파티클
  - 파티클 오버레이: 카드 밖으로 삐져나오는 이펙트
```

### 웹 구현
```
방법: CSS mask-image + Canvas 2D 오버레이
  - 정지 일러스트(PNG)를 <img>로 표시
  - 그 위에 <canvas>를 절대 배치 (pointer-events:none)
  - 마스크 PNG로 "움직일 영역"만 노출
  - @keyframes로 opacity/transform 루프

필요 에셋 (카드당):
  - 일러스트 PNG 1장 (기존 그대로)
  - 마스크 PNG 1장 (움직일 영역만 흰색) ← SD 2차 생성 또는 GIMP 수동
  - 공용 noise.png 1장 (모든 카드 공유)

총 추가 용량: 카드당 ~30KB (마스크 PNG, 512x512 알파)
대안: 마스크 없이 단순 CSS filter + opacity pulse만 쓰면 추가 용량 0
```
