# Realm of Fates — Game Design Document (GDD)
> 최종 업데이트: 2026-04-09

---

## 1. 게임 컨셉

### 한줄 요약
**당신은 신이다. 영웅과 유닛을 모아 신들의 리그에서 최고의 신이 되어라.**

### 장르
로그라이크 카드 오토배틀러 + 마을 경영 + 가챠

### 플랫폼
모바일 우선 웹앱 (PWA 가능)

### 타겟
캐주얼~미드코어 (한 판 3~10분)

### 핵심 재미
1. **성장 쾌감** — 브론즈 카드를 전설/신 등급으로 키우는 재미
2. **전략 배치** — 근접/원거리, 앞열/뒷열, 시너지 조합
3. **카드 수집** — 300+ 유닛, 100+ 스킬, 100+ 유물

---

## 2. 세계관 & 스토리

### 배경
신들의 영역. 수많은 신들이 존재하며, 각 신은 자신만의 마을과 영웅을 거느린다.
최고의 신이 되기 위해 **신들의 리그**에서 경쟁한다.

### 플레이어 역할
- 플레이어 = **신** (직접 싸우지 않음)
- **영웅 카드** = 신을 대리하는 전사
- **유닛 카드** = 마을에서 고용한 용병/병사

### 목표
신들의 리그에서 점수를 모아 **최고의 신** 칭호 획득

## 3. 핵심 시스템

### 3-1. 마을 시스템 (신규)
플레이어의 거점. 건물을 짓고 자원을 모으는 경영 요소.

| 건물 | 기능 | 해금 조건 |
|------|------|----------|
| **선술집** | 유닛 고용 (가챠) | 기본 |
| **훈련소** | 스킬 훈련 (가챠) | 리그 실버 |
| **대장간** | 장비 제작 (가챠/크래프팅) | 리그 골드 |
| **광산** | 시간당 골드 생산 | 마을 Lv2 |
| **성** | 퀘스트 수행, 수련 | 마을 Lv3 |
| **부활의 제단** | 죽은 유닛 부활 (비용) | 마을 Lv2 |
| **묘지** | 유닛 매장 → 명예 획득 | 기본 |
| **연구소** | 시너지/종족 연구 | 리그 골드 |

#### 퀘스트 시스템
- 성에서 퀘스트 수행 → 실시간 타이머
- 난이도별: 10분 / 30분 / 1시간 / 4시간 / 800분(13시간)
- 보상: 장비, 스킬, 특수 아이템, 골드, 경험치

#### 유닛 사망 시스템
- 리그 전투에서 유닛 사망 시 선택:
  - **부활** (골드 or 특수 아이템) → 유닛 복귀
  - **매장** (소량 골드) → 유닛 영구 삭제 + 명예 획득

### 3-2. 리그 시스템 (신들의 리그)

#### 리그 등급 & 점수
| 리그 | 필요 점수 | 보상 배율 |
|------|----------|----------|
| 브론즈 | 0~99 | x1 |
| 실버 | 100~299 | x1.5 |
| 골드 | 300~599 | x2 |
| 플래티넘 | 600~999 | x2.5 |
| 다이아 | 1000~1499 | x3 |
| 마스터 | 1500~1999 | x4 |
| 신의 영역 | 2000+ | x5 |

#### 점수 획득
| 결과 | 기본 점수 | 상위 상대 가산 |
|------|----------|--------------|
| 승리 | +10 | +5~20 (차이에 비례) |
| 패배 | +5 | - |

#### 매칭
- 유저 영웅 레벨 & 카드 레어리티 기반
- 같은 리그 내에서 ±1 티어 범위 매칭
- 비슷하지만 랜덤 요소 유지

#### 시즌 시스템
- 시즌 기간: 1개월 (또는 설정)
- 시즌 종료 시:
  - **최고의 신** 달성자 → 한정 일러스트 + 장비 + 스킬
  - 업적으로 영구 기록
  - 리그 점수 소프트 리셋 (50% 유지)

### 3-3. 아레나 시스템 (신규)

#### 컨셉
- **로테이션 챔프**만 사용 가능
- 매주/매 시즌 사용 가능 챔프가 바뀜
- 모든 플레이어가 같은 풀 → 순수 전략 경쟁

#### 랭킹
- 별도 아레나 랭킹
- 보상: 한정 일러스트, 장비, 스킬

### 3-4. 전투 시스템

#### 전투 플로우
```
출전 선택 (유닛 + 유물) → 배치 → 전투 시작
  → 라운드 1 (1번 공격) → 생존 → 4개 선택지 중 택1
  → 라운드 2 (2번 공격) → ...
  → 영웅 사망 → 승리/패배 → 로그라이크 리셋
```

#### 라운드 사이 선택지 (4개)
1. **유닛 선택** — 임시 유닛 추가 (중복=합성)
2. **스킬 선택** — 임시 스킬 장착 (중복=합성)
3. **유물 선택** — 임시 유물 적용 (중복=합성)
4. **체력 포션** — 영웅 HP 50% 회복
   - 사용 후 다음 라운드 사용 불가 (쿨다운 1라운드)
   - AI: 영웅 HP 30% 이하 시 자동 사용

#### 로그라이크 규칙
- 전투 중 얻은 모든 것(유닛/스킬/유물/강화) → 전투 끝나면 초기화
- 영구 보존: 골드, 경험치, 명예, 리그 점수

### 3-5. 카드 시스템

#### 카드 종류
| 종류 | 목표 수량 | 설명 |
|------|----------|------|
| 유닛 | 300 | 전투 유닛 |
| 스킬 | 100 | 유닛에 장착 |
| 유물 | 100 | 전체 버프 |
| 장비 | 100+ | 대장간 제작, 유닛 장착 |

#### 등급 체계
브론즈 → 실버 → 골드 → 전설 → 신(Divine)

#### 종족 & 시너지 ("운명의 왕좌" 고유 IP)
> 상세는 `PHASE2_SYNERGY_PLAN.md` 참조

| 종족 | 시너지 (2체) | 시너지 (4체) |
|------|------------|------------|
| 인간 (Human) | 전체 HP+10% | 전체 HP+25%, 방어+3 |
| 숲의 계약자 (Sylvan) | 스피드+15% | 스피드+30%, 회피+5 |
| 망자의 군단 (Wraith) | 흡혈 10% | 흡혈 25%, 부활 가능 |
| 원룡 (Drake) | 공격+15% | 공격+35%, 브레스 추가 |
| 심연의 권속 (Abyssal) | 치명타+10% | 치명타+25%, 저주 추가 |
| 광명의 사도 (Luminary) | 회복+20% | 회복+40%, 보호막 추가 |
| 태엽장치 (Clockwork) | 방어+15% | 방어+30%, 보호막 추가 |
| 야수 (Beast) | 야수당 공격+1 | 야수당 공격+2, HP+5 |

#### 전직 시스템 (신규)
- 유닛이 명예 일정량 모으면 **전직** 가능
- 예: 기사 → 성기사 / 암흑기사 (선택)
- 전직 시 스탯 대폭 상승 + 새 스킬 해금

### 3-6. 가챠 시스템 (신규)

#### 유닛 고용 (선술집)
| 등급 | 확률 | 골드 비용 |
|------|------|----------|
| 브론즈 | 50% | 5 |
| 실버 | 30% | - |
| 골드 | 15% | - |
| 전설 | 4% | - |
| 신 | 1% | - |
- 10연차 시 실버 이상 1개 보장
- 천장: 50회 시 전설 보장, 100회 시 신 보장

#### 스킬 훈련 (훈련소)
| 등급 | 확률 | 비용 |
|------|------|------|
| 브론즈 | 45% | 3 |
| 실버 | 30% | - |
| 골드 | 18% | - |
| 전설 | 6% | - |
| 신 | 1% | - |

#### 장비 제작 (대장간)
- 재료 + 골드로 제작
- 제작 결과 등급 = 재료 등급 + 운(행운 스탯 영향)

### 3-7. 과금 모델

| 상품 | 가격(안) | 내용 |
|------|---------|------|
| 골드 패키지 | $1~$10 | 골드 100~1500 |
| 프리미엄 가챠 티켓 | $3 | 10연차 1회 |
| 시즌 패스 | $5/월 | 일일 보상 2배, 퀘스트 단축 |
| 건설 즉시완료 | $0.5 | 대기 시간 스킵 |

## 4. UI/UX 방향

### 모바일 우선
- 세로 모드 최적화
- 큰 터치 타겟 (최소 44px)
- 스와이프 제스처

### 비주얼 스타일
- 다크 석재 고딕 아치 프레임 (05-design-direction.md 준수)
- 프레임리스 일러스트 (Chrono Knight 레퍼런스)
- SVG 아이콘 + CSS 프레임 (현재 임시)
- 풀 일러스트로 전환 중 (모든 유닛 PNG 보유)

### 주요 화면
```
메인 (마을 뷰)
├── 선술집 (가챠: 유닛)
├── 훈련소 (가챠: 스킬)
├── 대장간 (가챠/제작: 장비)
├── 광산 (자원 수집)
├── 성 (퀘스트)
├── 부활의 제단 / 묘지
├── 상태 확인 (카드/장비/스킬 관리)
├── 신들의 리그 (메인 전투)
├── 아레나 (로테이션 전투)
└── 상점 (과금)
```

## 7. 핵심 수치 (밸런스 시트)

### 경제 (Phase 1 5-1 밸런스 확정, 2026-04-11)
| 항목 | 값 |
|------|-----|
| 초기 골드 | **5** |
| 리그 승리 | +**12** 골드, +10 XP, +10 명예, +10 리그점수 |
| 리그 패배 | +**5** 골드, +10 XP, +1 명예, +5 리그점수 |
| 유닛 가챠 1회 | 5 골드 |
| 스킬 가챠 1회 | 3 골드 |
| 카드 강화 (단련) | 레벨×**2+1** 골드 |
| 선술집 새로고침 | **1** 골드 |
| 유닛 부활 | 10~50 골드 (등급별) |
| 유닛 매장 | 3 골드 → +5~20 명예 |

### 전투
| 항목 | 값 |
|------|-----|
| 라운드 N | N번 공격 |
| 평균 전투 길이 | 3~7 라운드 |
| 포션 회복량 | HP 50% |
| 포션 쿨다운 | 1라운드 |
| 출전권 | 영웅 레벨 |
| 유물력 | 영웅 레벨/2 |

### 카드 레벨업
| 레벨 | 필요 XP |
|------|---------|
| 1→2 | 10 |
| 2→3 | 20 |
| 3→4 | 30 |
| N→N+1 | N×10 |
