# Phase 2-2: 카드 플레이버 텍스트 계획

> 목표: 모든 카드에 세계관 몰입형 한줄 문구 추가 (업계 표준 카드게임 몰입 요소)

---

## 현재 상태
- 영웅 카드 18종 + 모집 유닛 24종 = 42종에 이미 `flavor` 필드 존재 ✅
- 스킬 29종, 유물 12종에는 flavor 없음

## 작업 범위

### 1. 기존 유닛 플레이버 확인 (42종) ✅
이미 모든 유닛에 flavor가 있음. 예시:
- 민병: *"쟁기를 내려놓고 검을 들었다."*
- 드래곤: *"태고의 분노가 깨어날 때, 세계는 침묵한다."*
- 대천사: *"천상의 날개가 펼쳐지면, 어둠은 물러간다."*

### 3. 유물 플레이버 추가 (12종)
> 유물 데이터 확인 후 작성 필요

### 4. 구현 시 고려사항
- 카드 하단에 이탤릭체로 표시 (업계 표준 플레이버 텍스트 배치)
- 폰트: 세리프 계열 (Georgia, Noto Serif KR)
- 색상: `#888` 또는 `#a09070` (은은한 갈색)
- 카드 상세 보기에서만 표시 (미니 카드에선 생략)
- 카드 뒤집기/획득 시 플레이버 텍스트 fade-in 연출
