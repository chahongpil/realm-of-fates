## 구현 계획 (8단계)

### 1단계: KEYWORDS 데이터에 tier + element 추가 ✅ (사전 완료, tier/element 추가 필요)
- [ ] 기존 KEYWORDS에 `tier:'bronze'|'silver'|'gold'|'legendary'|'divine'` 필드 추가
- [ ] `element:'fire'|'water'|'lightning'|'earth'|'dark'|'holy'|null` 필드 추가

### 2단계: 키워드 CSS (등급별 + 원소별)
- [ ] `.kw` — 등급별 텍스트 색상 (기본은 금색, tier에 따라 변화)
- [ ] `.kw-tooltip` — 기본 다크 프레임
- [ ] `.kw-tooltip.tier-bronze` ~ `.tier-divine` — 등급별 테두리 + shadow
- [ ] `.kw-tooltip.elem-fire` ~ `.elem-holy` — 원소별 배경 그라데이션
- [ ] `.kw-tooltip .kw-sigil` — 신의 인장 워터마크 (opacity: 0.04~0.07)
- [ ] `.kw-tooltip .kw-tier-badge` — 등급 뱃지 (좌상단 작은 아이콘)
- [ ] 등장 애니메이션: fade-in + 슬라이드
- [ ] 모바일/반응형 대응

### 3단계: 신의 인장 SVG/이모지 정의
- [ ] 6원소 신 인장 정의 (이모지 또는 CSS shape로 간단하게)
  - 🔥→🜂 / 💧→🜄 / ⚡→🜁 / 🌿→🜃 / 🌑→☽ / ✨→☀
  - CSS `::after` + opacity 0.05로 툴팁 배경에 삽입
- [ ] 크기: 툴팁의 60~80%, 중앙 정렬

### 4단계: 키워드 호버 사운드 (Web Audio)
- [ ] SFX에 키워드 사운드 5종 추가
  - `kw_bronze`: 가벼운 종이 넘기는 소리 (짧은 noise burst)
  - `kw_silver`: 은색 금속 울림 (mid sine)
  - `kw_gold`: 황금 종소리 (high sine + harmonics)
  - `kw_legendary`: 웅장한 울림 (low chord)
  - `kw_divine`: 뇌성 같은 짧은 이펙트 (bass + distortion)
- [ ] 호버 시 해당 등급 사운드 재생 (debounce 200ms)

### 5단계: parseKeywords() + 툴팁 이벤트
- [ ] `parseKeywords(text)` — `**keyword**` → `<span class="kw" data-kw="키워드명">` 변환
- [ ] `showKwTooltip(kwName, targetEl)` — 등급/원소 기반 툴팁 생성
  - KEYWORDS에서 조회 → tier/element 결정
  - 위치 계산 (위쪽 우선, 화면 밖이면 아래로)
  - 원소 배경 + 신의 인장 + 등급 테두리 적용
- [ ] `hideKwTooltip()` — 제거
- [ ] 이벤트 위임: `mouseover`/`mouseout` + 모바일 `click`

### 6단계: 카드 렌더링 적용
- [ ] `mkCardEl()` — skillDesc/bonusTrigger.desc에 parseKeywords() 적용
- [ ] `mkRelicEl()` — 유물 설명에도 적용
- [ ] 카드 상세 모달에도 적용

### 7단계: 전투 카드 키워드 뱃지
- [ ] 전투 카드 하단에 주요 키워드 뱃지 (최대 3개)
- [ ] 뱃지 색상 = 키워드 등급 색상
- [ ] 뱃지 호버 시에도 툴팁 표시
- [ ] 키워드 발동 시 뱃지 번쩍임 (flash animation)

### 8단계: 키워드 발동 연출 + 선택지 표시
- [ ] 전투 중 키워드 발동 시 배너 0.8초 표시 (등급 색상)
- [ ] 전투 로그에 키워드 아이콘 포함
- [ ] 라운드 선택지에서 유닛/스킬 키워드 뱃지 표시

## 툴팁 시각 상세 (등급별)

### 브론즈 등급 툴팁
```
┌─────────────────────────┐  ← 동색(#cd7f32) 테두리
│ 🥉 🛡️ 도발              │  ← 등급뱃지 + 아이콘 + 이름 (동색)
│                         │
│ 적이 이 유닛을 우선      │  ← 설명 (베이지)
│ 공격해야 합니다.         │
│            [인장 워터마크]│  ← opacity 0.05
└─────────────────────────┘
```

### 전설 등급 툴팁
```
╔═════════════════════════╗  ← 보라색(#a335ee) 테두리 + glow
║ 👑 💀 사령술             ║  ← 등급뱃지 + 아이콘 + 이름 (보라)
║                         ║
║ 처치한 적을 아군         ║  ← 설명
║ 언데드로 소환합니다.     ║
║      [암흑신 인장 + 안개]║  ← 보라 안개 배경 + 인장
╚═════════════════════════╝
```

### 신 등급 툴팁
```
╔═══════✦═══════╗  ← 빨간색(#ff4444) 맥동 glow
║ ⚡ ⚡ 뇌신의 심판║  ← 빨간 + 번쩍임
║               ║
║ 모든 적을     ║
║ 심판합니다.   ║
║  [뇌신 인장+번개]║  ← 번개 파티클 + 인장
╚═══════✦═══════╝
```

## 구현 순서 (의존성 기반)

```
1단계 KEYWORDS 데이터 보강 ──┐
                             ├── 5단계 파싱+툴팁 ── 6단계 카드적용 ── 7단계 전투뱃지 ── 8단계 발동연출
2단계 CSS ───────────────────┤
3단계 신의 인장 ─────────────┘
4단계 사운드 ────────────────┘
```

1~3단계는 병렬 작업 가능, 4단계도 독립적.
5단계부터 순차 진행.
