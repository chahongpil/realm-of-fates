## 설계 원칙: 시너지 65 : 파워 35 (StS 황금비)

- **시너지 의존 유닛 65%**: 단독 시 평균~약함, 조합 시 폭발 (현재 30% 수준 → 확대 필요)
- **파워 카드 35%**: 단독으로도 강함, 초보자/수집 욕구 대응
- **왜?**: 시너지 비율이 높을수록 덱빌딩 다양성 ↑. 파워 카드만 있으면 매 런이 똑같음.
- **재조정 방향**: 기존 유닛 중 일부 스탯을 **약간 낮추고** 시너지 조건부 버프를 **더 강하게**
  - 예: 늑대 현재 단독 HP 10 → 8로 낮춤, 대신 야수 2체 시 +3 HP / 4체 시 +5 HP
  - 예: 드래곤 스탯 유지하되, 원룡 2체 시 "브레스 공격" 추가

## 종족별 세계관 설명 (UI 툴팁용)
- **숲의 계약자**: 태고의 원소신과 계약을 맺은 원주민. 빠른 발걸음과 자연의 가호.
- **망자의 군단**: 죽음의 신 암흑이 일으킨 영원한 전사들. 피를 마시며 되살아남.
- **원룡**: 6원소 창조 이전부터 존재한 고대 용족. 순수한 공격 본능.
- **심연의 권속**: 차원 틈에서 건너온 그림자 존재. 치명타와 저주의 달인.
- **광명의 사도**: 빛의 신 신성에게 축복받은 사도들. 치유와 수호.
- **태엽장치**: 고대 문명이 남긴 마법 기계병. 철벽 방어.
- **야수**: 땅의 숨결을 이어받은 동료들. 무리 짓는 힘.

## UI 디자인

### 1. 전투 화면 상단 시너지 바

```
┌──────────────────────────────────────┐
│ 🐺 야수 ×2 [██]  ⚔️+2              │  ← 활성 시너지 (빛남)
│ 👤 인간 ×1 [█░]  (2체 필요)         │  ← 미달 시너지 (어두움)
└──────────────────────────────────────┘
```

- **위치**: 전투 화면 상단, HP바 아래
- **크기**: 작은 아이콘 + 텍스트 (전투 방해 안 되는 선)
- **활성**: 밝은 색 + glow, 종족 아이콘 빛남
- **미달**: 회색 + 프로그레스바로 몇 체 더 필요한지 표시

### 2. 편성 화면 시너지 미리보기

유닛 배치 시 **어떤 시너지가 발동/해제되는지** 실시간 표시:

```
┌─ 출전 편성 ──────────────────┐
│ [영웅⭐] [늑대🐺] [사냥꾼🏹] │
│                              │
│ 활성 시너지:                  │
│  🐺 야수 ×1 → +늑대 1마리 시 │
│     야수 ×2 시너지 발동!      │  ← 녹색 하이라이트
│  👤 인간 ×2 → HP +10% ✅     │
└──────────────────────────────┘
```

### 3. 시너지 발동 연출

- **2체 달성**: 시너지 아이콘이 번쩍 + "야수 시너지 발동!" 텍스트 0.8초
- **4체 달성**: 황금 폭발 이펙트 + "야수 완전 시너지!" + 전용 사운드
- 발동 시 해당 종족 유닛들에 테두리 하이라이트

### 4. 시너지 상세 팝업

시너지 아이콘 클릭/호버 시 상세 팝업:

```
╔════════════════════════╗
║ 🐺 야수의 결속          ║
║                        ║
║ ×2: 야수 수×공격+1     ║  ← 현재 활성 (밝음)
║ ×4: 야수 수×공격+2,HP+5║  ← 미달 (어두움)
║                        ║
║ 보유 야수: 늑대, 썬더버드 ║
╚════════════════════════╝
```

## 웹 구현 시 고려사항

- **TFT/오토체스 참조**: 화면 좌측에 세로 시너지 바 (`<div class="synergy-bar">`)
- **애니메이션**: CSS `@keyframes` + 클래스 토글 (시너지 발동 시 `.active` 추가)
- **사운드**: 종족별 차등 사운드 (Web Audio) — 야수=울부짖음, 원룡=포효 등
- **파티클**: 시너지 활성 시 Canvas 2D 파티클 (원소 테마: 화염/빙결/번개)
