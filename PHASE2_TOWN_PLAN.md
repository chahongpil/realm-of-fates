# Phase 2: 마을 시스템 종합 기획

> 목표: 45도 쿼터뷰 탑다운 마을 + 건물별 기능 완성 + 생활 애니메이션
> 마지막 업데이트: 2026-04-11

---

## 1. 45도 쿼터뷰 마을 비주얼 디자인

### 핵심 컨셉
- **시점**: 45도 아이소메트릭 쿼터뷰 (위에서 내려다보기)
- **배경**: 2.5D (2D 일러스트 배경 + 3D 건물 오브젝트)
- **카메라**: 건물 탭 → 자동 줌인 (Cinemachine)
- **건물 클릭**: 줌인(0.5초) → 건물 문 열림 → NPC 대화 화면
- **하단 퀵메뉴**: 건물 8개 아이콘 바 (모바일 필수)
- **건설 전**: 빈 터 (폐허/잔디/바위)
- **건설 시**: 건축 애니메이션과 함께 건물이 **뿅!** 올라옴
- **마을 생활**: 건물마다 NPC들이 살아 움직이는 애니메이션
- 배경은 시간대/날씨/건물 레벨에 따라 동적 변화

### 마을 화면 레이어 구조
```
┌─────────────────────────────────┐
│  [HUD: 💰1234 💎56 ✨12]        │ ← UI 레이어 (최상단, 3종 화폐)
│                                 │
│    ⛰️ 산맥 (2D 일러스트)         │ ← 배경 레이어
│                                 │
│      🏰  📋  ⛪                  │
│    ⚒️    ⛲    🏪                │ ← 건물 레이어 (3D/2.5D)
│      🍺  🏟️  🚪                 │
│                                 │
│    🌳  행인NPC  🐦  🌳          │ ← NPC/소품 레이어
│                                 │
├─────────────────────────────────┤
│ 🏰  🚪  ⚒️  🏪  🍺  🏟️  📋  ⛪ │ ← 하단 퀵메뉴
└─────────────────────────────────┘
```

### 건물 클릭 흐름
```
마을 화면 (전체 뷰)
    ↓ 건물 탭 or 퀵메뉴 탭
카메라 줌인 (0.5초, Cinemachine)
    ↓
건물 문 열림 애니메이션
    ↓
NPC 대화 화면 (화면 절반 NPC 일러스트 + 선택지)
    ↓ [돌아가기]
카메라 줌아웃 → 마을 전체 뷰
```

### NPC 대화 화면 (비주얼 노벨 스타일)
```
┌─────────────────────────────────┐
│                                 │
│  [NPC 일러스트]    [대사 텍스트]  │
│  (화면 좌측 절반)   "어서 와!"   │
│                                 │
│                   [🗡️ 용병 모병] │
│                   [✨ 영웅 고용] │
│                   [← 돌아가기]  │
│                                 │
└─────────────────────────────────┘

NPC 특징:
- 같은 인물이 레벨에 따라 성장 (의상/배경 변화)
- 여성 NPC: 선술집/성/상점/교회/서고 (5명)
- 남성 NPC: 대장간/훈련장/성문 (3명)
```

### 마을 레이아웃 (45도 쿼터뷰)

```
                    ⛰️ 산맥 배경 ⛰️
              ╱‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾╲
           ╱    🏰 성 (최상단)      ╲
          │   ╱              ╲      │
         ╱  🚪성문          📋서고   ╲
        │  ╱     ╲      ╱     ╲     │
       ╱ ⚒대장간  ⛲광장  🏪상점     ╲     ← 석재 도로
      │           │                  │
     ╱  🍺선술집  🏟훈련장  ⛪교회     ╲
    │     🌳    🌳    🌳    🌳       │
    ╲═══════════ 성벽/정문 ═══════════╱
          🌲  🌲  길  🌲  🌲
```

- 성이 가장 높은 곳 (뒤쪽 = 위쪽)에 위치
- 아래로 갈수록 가까운 건물 (원근감)
- 건물 사이에 **석재 도로**가 연결 (건설하면 도로도 깔림)
- 중앙에 **광장 + 분수** (마을 심장부, 주민 모이는 곳)
- 줌인/줌아웃 지원 (줌인 → 건물 상호작용, 줌아웃 → 전체 조감)

## 2. 건물별 생활 애니메이션

> 각 건물 주변에서 NPC들이 살아 움직이는 애니메이션. 건물 레벨이 올라갈수록 애니메이션 종류와 NPC 수가 증가.

### 🏰 성 — 통치의 심장

| Lv | 애니메이션 |
|----|----------|
| 1 | 늙은 조언자가 문 앞에서 빗자루질 |
| 2 | 하인 1명이 물통을 나름, 깃발이 바람에 펄럭임 |
| 3 | 근위병 2명이 성문 좌우에서 경비, 전령이 뛰어다님 |
| 4 | 기사가 말을 타고 성 주변 순찰, 시녀가 창문에서 내다봄 |
| 5 | 왕의 행렬 (호위병 + 깃발), 성 꼭대기에서 독수리 날아다님, 왕좌 앞 카펫에 청원자 줄 서기 |

**상시**: 성 굴뚝에서 연기, 깃발 펄럭임, 창문 불빛 (밤)

### 🚪 성문 — 출전의 관문

| Lv | 애니메이션 |
|----|----------|
| 1 | 소년 파수꾼이 졸면서 고개를 끄덕임 |
| 2 | 문지기가 서성이며 정찰, 가끔 하품 |
| 3 | 근위병 2명이 교대 의식 (제자리 행진 후 교체) |
| 4 | 정찰대가 성문 밖으로 나갔다 돌아옴, 전령 비둘기 날아감 |
| 5 | 성문이 주기적으로 열렸다 닫힘, 마차/행상이 드나듦, 깃발 근위대 정렬 |

**전투 출전 시**: 성문이 활짝 열리며 영웅이 말 타고 나가는 연출

### ⚒️ 대장간 — 불꽃의 작업장

| Lv | 애니메이션 |
|----|----------|
| 1 | 견습공이 서툴게 모루를 두드림 (가끔 실수해서 화들짝) |
| 2 | 대장장이가 힘차게 망치질, 불꽃 튐, 화로에서 연기 |
| 3 | 도제 1명 추가 — 풀무질(바람 불어넣기), 대장장이 담금질 (쇠를 물에 넣으면 증기 파티클) |
| 4 | 마법 룬이 장비에 새겨지는 빛 이펙트, 마법대장장이가 주문 시전 |
| 5 | 용광로에서 신의 불꽃 (파란불), 완성된 검이 빛나며 공중에 뜸, 도제 3명이 일하는 분주한 작업장 |

**상시**: 화로 불빛 흔들림, 망치 소리 (근처 가면 들림), 연기 파티클

### 🏪 상점 — 활기찬 장터

| Lv | 애니메이션 |
|----|----------|
| 1 | 행상인 할머니가 보따리 펼치고 손짓으로 호객 |
| 2 | 손님 1~2명이 물건 구경, 상인이 물건 진열 |
| 3 | 행상 마차 도착 → 짐 내리기, 손님 3~4명 북적 |
| 4 | 이국적 상인(터번 쓴 사람)이 보석/양탄자 펼침, 아이가 사탕 사먹기 |
| 5 | 대형 시장 — 천막 여러 개, 다양한 상인들, 저울질하는 모습, 호객 소리 |

**상시**: 천막이 바람에 흔들림, 물건이 반짝반짝

### 🍺 선술집 — 용병들의 아지트

| Lv | 애니메이션 |
|----|----------|
| 1 | 주막주인이 술통 닦기, 취객 1명이 비틀비틀 |
| 2 | 용병 2명이 탁자에서 카드 놀이, 주모가 맥주 서빙 |
| 3 | 음유시인이 류트 연주 (음표 파티클♪), 손님 5명+ 북적, 건배 |
| 4 | 팔씨름 대회 (용병 2명, 구경꾼 환호), 2층 발코니에서 누군가 내다봄 |
| 5 | 전설 영웅의 조각상 앞에서 신입 용병이 경의, 화려한 연회 분위기, 불 삼키는 곡예사 |

**상시**: 문 열릴 때 불빛+웃음소리 새어나옴, 간판 삐걱삐걱 흔들림

### 🏟️ 훈련장 — 전사들의 땀

| Lv | 애니메이션 |
|----|----------|
| 1 | 허수아비 1개, 신참 1명이 허수아비를 서툴게 공격 (빗맞추기도 함) |
| 2 | 검술교관이 신참 교정 (자세 잡아주기), 나무검 스파링 |
| 3 | 전사 4~5명이 줄 서서 동시에 검 휘두르기, 사수가 과녁 사격, 활시위 소리 |
| 4 | 기마 훈련 (말 위에서 검술), 모래먼지 파티클, 관중석에서 장군이 팔짱끼고 관찰 |
| 5 | 콜로세움 — 2명이 실전 스파링 (공격/방어 교환), 관중 NPC 환호, 훈련 후 악수, 횃불 조명 |

**상시**: 훈련 구호 소리 ("하! 하!"), 먼지, 함성

### 📋 서고 — 지식의 전당

| Lv | 애니메이션 |
|----|----------|
| 1 | 서생이 책 읽다 졸기 (고개 까딱까딱) |
| 2 | 학자가 깃펜으로 뭔가 기록, 책장에서 책 꺼내기 |
| 3 | 현자와 제자가 토론 (말풍선 번갈아 표시), 책이 쌓인 탁자 |
| 4 | 마법 수정구 빛남, 대현자가 공중에 마법 문자 그리기 |
| 5 | 거대 도서관 — 책이 자동으로 날아서 정리됨, 마법 빛줄기가 책장 사이로, 다수의 학자 연구 |

**상시**: 책장 먼지 반짝임, 촛불 흔들림, 조용한 분위기

## 3. 공용 마을 애니메이션

### 광장 (중앙)

| 마을 총 레벨 | 광장 모습 |
|-------------|----------|
| 0~10 | 빈 공터, 잡초, 깨진 분수 |
| 11~20 | 분수 복구 (물 졸졸), 벤치 1개, 주민 2~3명 |
| 21~30 | 시장 천막, 아이들 뛰어다님, 분수 앞 연인 |
| 31~40 | 화려한 분수 (물줄기 높이↑), 거리 악사, 행상, 깃발 장식, 축제 분위기 |

### 도로 위 행인 NPC

건물 레벨에 따라 도로 위를 걸어다니는 NPC:

| NPC 유형 | 행동 | 등장 조건 |
|---------|------|----------|
| 농부 | 수레 끌고 이동 | 기본 |
| 아이들 | 뛰어다니며 놀기 | 총 Lv 10+ |
| 상인 | 짐 지고 상점 ↔ 성문 이동 | 상점 Lv2+ |
| 순찰병 | 성벽 따라 걷기 | 성문 Lv3+ |
| 기사 | 말 타고 순찰 | 성 Lv4+ |
| 전령 | 뛰어다니며 두루마리 전달 | 서고 Lv3+ |
| 수도승 | 교회 ↔ 서고 이동 | 교회 Lv2+ |
| 도제 | 재료 지고 대장간으로 | 대장간 Lv2+ |
| 취객 | 선술집에서 비틀비틀 나옴 | 선술집 Lv2+ |
| 음유시인 | 광장에서 연주 | 선술집 Lv3+ |
| 부상병 | 교회로 절뚝절뚝 | 전투 패배 후 |
| 전사 | 훈련장 ↔ 선술집 이동 | 훈련장 Lv2+ |

## 4. 건설 애니메이션 (뿅! 효과)

### 쿼터뷰 건설 시퀀스 (2초)

```
0.0s: 빈 터에서 먼지구름 뿜어올라감 (파티클)
0.3s: 땅이 살짝 흔들리는 카메라 쉐이크
0.5s: 건물 기초(돌)가 아래에서 위로 올라옴 (DOTween MoveY)
0.8s: 건물 벽체가 조립됨 (스케일 0→1, 약간 탄력 있게 bounce)
1.0s: 지붕이 올라앉음 + 망치 소리 효과음
1.2s: 간판/깃발/디테일 등장 (fade-in)
1.5s: 금빛 빛줄기가 건물을 감쌈 (glow 파티클)
1.8s: NPC가 문을 열고 나옴 (첫 인사 대사 말풍선)
2.0s: 건물 주변 생활 애니메이션 시작
```

### 레벨업 시퀀스 (1.5초)

```
0.0s: 건물에 빛의 기둥이 솟음
0.3s: 건물이 살짝 투명해지며 기존 모델 fade-out
0.5s: 새 레벨 건물 모델이 fade-in (크기 약간 커짐)
0.8s: 확장된 부분 (탑, 2층 등) bounce 등장
1.0s: 등급 폭발 이펙트 (키워드 시스템 등급 색상)
1.2s: NPC가 새 모습으로 교체, 업그레이드 대사
1.5s: 새 생활 애니메이션 추가 NPC 등장
```

## 5. 건물별 기능 상세 (미구현 건물)

### ⚒️ 대장간 — 장비 시스템

#### 장비 슬롯 (유닛당 3개)
| 슬롯 | 종류 | 효과 예시 |
|------|------|----------|
| 무기 | 검/활/지팡이/도끼 | 공격력/치명타/관통 등 |
| 방어구 | 갑옷/로브/가죽 | HP/방어력/마법방어 등 |
| 장신구 | 반지/목걸이/부적 | 스피드/회피/에너지 등 |

#### 제작 시스템
```
재료 + 골드 → 장비

재료 획득: 전투 드랍 / 장비 분해 / 퀘스트 / 상점
제작 등급: 재료 등급 + 행운 스탯 → 결과 등급

제작 UI:
╔═ ⚒️ 대장간 ════════════════════╗
║ [제작]  [강화]  [분해]          ║
║                                ║
║ 📦 재료: 철광석 ×3, 가죽 ×2    ║
║ 💰 비용: 15골드                 ║
║ 🎲 예상 등급: 실버~골드          ║
║                                ║
║ ⚔️ 강철검 (실버)               ║
║   공격 +4, 치명타 +3%          ║
║                                ║
║ [제작하기]                      ║
╚════════════════════════════════╝
```

## 7. 건물 의존성 트리

```
               🏰 성 (마스터 건물)
              ╱  │  ╲
           ╱     │     ╲
        ╱        │        ╲
  🚪 성문    📋 서고    ⛪ 교회
       │                    │
  🏟️ 훈련장              ⚒️ 대장간
                              │
                          🏪 상점
                          
  🍺 선술집 (독립)
```

**규칙**: 다른 건물의 레벨은 성 레벨을 초과할 수 없음
**추가 규칙**:
- 대장간 Lv3+ → 상점에서 재료 판매 시작
- 서고 Lv4 (모의전투) → 훈련장과 연동
- 교회 Lv5 (무료치료) → 전투 후 자동 치료 옵션

│   └── .building[data-type="church"]
├── .npc-layer

│   └── .plaza-npc
