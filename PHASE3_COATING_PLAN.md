> 목표: 업계 표준 카드 메탈릭 변형 스킨 + 수집 욕구 자극. 순수 시각적 보상 시스템.
> 레퍼런스 연구: `.claude/references/card-game-studies/`

## 코팅 등급 (5단계) — 전 기획서 공용 명칭

> ⚠️ 코팅 명칭은 반드시 아래 5종만 사용. "브론즈 코팅", "실버 코팅" 등은 사용 금지.

| 등급 | 코팅명 | 시각 효과 | 획득 방법 |
|------|--------|----------|----------|
| 1 | **일반** | 기본 일러스트 | 기본 |
| 2 | **빛나는** (홀로그램) | 카드에 홀로그램 반짝임 | 실버~골드 리그 시즌 보상 / 10연승 / 업적 |
| 3 | **황금** | 황금 테두리 + 일러스트 파티클 | 플래티넘~다이아 리그 시즌 보상 / 가챠 |
| 4 | **신화** | 특수 오라 + 전용 파티클 | 마스터~신의영역 시즌 보상 / 한정 이벤트 |
| 5 | **태고** | 고대 문양 + 원소 파티클 | 최고 희귀 (확률 0.1% 가챠) / 절대신 클리어 |

> "신의 축복"은 **화폐 이름**(축복)으로만 사용. 코팅 4등급은 **"신화"**로 통일.

## 시각 효과 상세

### 일반 (Normal)
- 기본 카드 디자인 그대로
- 변경 없음

### 빛나는 (Holographic)
```
효과:
- 카드 기울기에 따라 무지개빛 반사 (CSS perspective + gradient)
- 은은한 홀로그램 줄무늬가 카드 표면에 흘러감
- 카드 테두리에 미세한 반짝임

웹 구현:
- CSS `linear-gradient(135deg, ...)` + `background-position` 애니메이션
- 카드에 `.coating-holo` 클래스 추가 → `::after` pseudo로 홀로그램 오버레이
- `@keyframes holoShift` 60s 반복
```

### 황금 (Golden)
```
효과:
- 카드 전체가 황금빛 색조 (메탈릭 변형 스킨)
- 일러스트 내부에 파티클 움직임 (불 유닛이면 불꽃, 물 유닛이면 물결)
- 카드 테두리 금빛 + glow
- 뒤집을 때 금빛 트레일

웹 구현:
- CSS `filter: sepia(1) hue-rotate(-20deg) saturate(3)` + 황금 테두리
- `.coating-gold` 클래스 + Canvas 2D 파티클 레이어 (원소별)
- 뒤집기 시 `@keyframes goldTrail` + 금빛 box-shadow
```

### 신화 (Mythic)
```
효과:
- 카드 주변에 원소 오라 회전
- 일러스트가 살짝 움직이는 느낌 (parallax)
- 특수 사운드 (카드 호버 시)
- 카드 뒷면도 특수 디자인 (신의 인장)

웹 구현:
- Canvas 2D 파티클로 원소별 오라 (fire=불꽃 회전, water=물결 등)
- CSS `transform: translate(calc(var(--mx)*2px), ...)` 마우스 포지션 parallax
- Web Audio API 카드별 전용 사운드 재생
```

### 태고 (Ancient)
```
효과:
- 고대 문양이 카드 테두리에 새겨짐 (룬 문자 느낌)
- 원소 파티클이 카드 밖으로 흘러나옴
- 카드를 올리면 배경이 살짝 어두워짐 (스포트라이트)
- 전투 중 공격 시 특수 이펙트

웹 구현:
- SVG 룬 문양 오버레이 + CSS `@keyframes runeGlow` (opacity 0.3~1)
- Canvas 2D 파티클을 카드 bounding box 밖으로 확산
- 카드 선택 시 전체 화면 `backdrop-filter: brightness(0.6)` vignette
```

## 획득 방법 상세

### 빛나는 (Holographic)
| 방법 | 조건 | 대상 |
|------|------|------|
| 10연승 | 10연승 달성 시 | 가장 많이 출전한 유닛 1장 |
| 업적 | 특정 업적 달성 | 지정된 유닛 |
| 단련 | 같은 유닛 Lv.20 달성 | 해당 유닛 |

### 황금 (Golden)
| 방법 | 조건 | 대상 |
|------|------|------|
| 시즌 보상 | 골드 리그 이상으로 시즌 마감 | 가장 많이 사용한 유닛 1장 |
| 가챠 | 황금 티켓 사용 | 랜덤 |
| 전직 | 골드→전설 전직 시 | 해당 유닛 |

### 신화 (Mythic)
| 방법 | 조건 | 대상 |
|------|------|------|
| 신의영역 달성 | 리그 2000점 달성 | 영웅 카드 |
| 한정 이벤트 | 시즌 한정 이벤트 | 이벤트 보상 유닛 |
| 100연승 | 누적 100연승 | 선택 유닛 1장 |

### 태고 (Ancient)
| 방법 | 조건 | 대상 |
|------|------|------|
| 극히 희귀 가챠 | 0.1% 확률 | 랜덤 |
| 절대신 클리어 | 최종 보스 클리어 | 영웅 카드 |

## 코팅 UI

### 카드 컬렉션에서 코팅 표시
```
┌─────────────┐
│ 🐺 늑대      │
│ [일러스트]    │
│              │
│ ✨ 빛나는     │  ← 코팅 뱃지
│ 보유 코팅: 2/5│
└─────────────┘
```

### 코팅 변경 화면
```
╔═ 카드 코팅 변경 ═══════════════╗
║ 🐺 늑대                        ║
║                                ║
║ [일반]  [✨빛나는]  [🔒황금]     ║
║  ✅      ✅         🔒         ║
║                                ║
║ 현재 장착: ✨ 빛나는            ║
║ [일반으로 변경]  [빛나는 유지]   ║
╚════════════════════════════════╝
```

### 코팅 획득 연출
1. 화면 중앙에 카드 등장
2. 기존 카드가 빛나며 분해
3. 새 코팅이 적용된 카드가 조립되며 등장
4. 등급에 맞는 파티클 폭발
5. "✨ 황금 코팅 획득!" 배너

## 웹 구현 시 고려사항
- **CSS 레이어**: 각 코팅별 전용 클래스 (`.coating-holo`, `.coating-gold`, `.coating-mythic`, `.coating-ancient`)
- **파티클**: Canvas 2D 공유 파티클 풀, 코팅별 프리셋 JSON
- **데이터**: `save.ownedCoatings = { "h_m_fire": ["gold", "mythic"], ... }` 객체
- **성능**: 황금/신화/태고는 CSS filter 비용이 높으므로, 한 화면 최대 **10장 동시 표시** 제한
