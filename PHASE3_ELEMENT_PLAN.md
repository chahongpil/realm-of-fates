## 원소 상성표

### 4원소 순환 + 빛/어둠 대립

```
        🔥 화염
       ↗     ↘
  🌿 대지     💧 물
       ↖     ↙
        ⚡ 번개

  ✨ 신성 ↔ 🌑 암흑 (상호 상극)
```

| 공격 원소 | 유리 (1.5배) | 불리 (0.75배) |
|----------|-------------|--------------|
| 🔥 화염 | 🌿 대지 | 💧 물 |
| 💧 물 | 🔥 화염 | ⚡ 번개 |
| ⚡ 번개 | 💧 물 | 🌿 대지 |
| 🌿 대지 | ⚡ 번개 | 🔥 화염 |
| ✨ 신성 | 🌑 암흑 | (자신 제외 없음) |
| 🌑 암흑 | ✨ 신성 | (자신 제외 없음) |

### 동일 원소
- 같은 원소끼리: **1.0배** (보너스/감소 없음)

## 전투 적용

### 데미지 계산 수정
```
기존: 피해 = 공격력 - 방어력
변경: 피해 = (공격력 × 원소배율) - 방어력

원소배율:
  유리: 1.5배 → 데미지 표시 색상 주황 + "유리!" 텍스트
  불리: 0.75배 → 데미지 표시 색상 파랑 + "저항!" 텍스트
  중립: 1.0배 → 기본 흰색
```

### 전투 UI 표시

#### 공격 시 원소 상성 표시
```
  ⚔️ 화염술사 🔥 → 늑대 🌿
     12 피해! (🔥→🌿 유리! ×1.5)    ← 주황색 + 크게 표시

  ⚔️ 화염술사 🔥 → 빙결술사 💧
     4 피해 (🔥→💧 저항 ×0.75)      ← 파란색 + 작게 표시
```

#### 편성 화면 상성 미리보기
적 영웅의 원소가 표시되므로, 유리한 원소 유닛을 편성하도록 유도:

```
┌─ 출전 편성 ──────────────────────┐
│ 적 영웅: 🔥 불의 전사             │
│ ────────────────────────         │
│ 💧 물 유닛 추천! (1.5배 피해)     │  ← 유리 원소 강조
│ 🌿 대지 유닛 주의! (0.75배 피해)  │  ← 불리 원소 경고
│                                  │
│ 보유 유닛:                        │
│  💧빙결술사 [추천✅] 🔥민병 [보통] │
│  🌿늑대 [주의⚠️] ⚡창병 [보통]    │
└──────────────────────────────────┘
```

## 원소 공명 (보너스 시스템)

같은 원소 유닛을 여러 체 편성하면 공명 보너스:

| 같은 원소 수 | 공명 효과 |
|-------------|----------|
| 2체 | 해당 원소 스킬 데미지 +10% |
| 3체 | 해당 원소 스킬 데미지 +20%, 원소 저항 +10% |
| 4체+ | 해당 원소 스킬 데미지 +35%, 원소 저항 +20%, 특수 효과 |

### 원소별 4체 특수 효과
| 원소 | 특수 효과 |
|------|----------|
| 🔥 화염 | "화염폭풍": 매 라운드 시작 시 적 전체 2 화상 |
| 💧 물 | "빙하의 축복": 아군 전체 스피드 +3 |
| ⚡ 번개 | "번개의 각성": 아군 전체 선제공격 1턴 |
| 🌿 대지 | "대지의 방벽": 아군 전체 방어 +3 |
| ✨ 신성 | "천상의 빛": 아군 전체 매턴 HP +2 |
| 🌑 암흑 | "암흑의 장막": 적 전체 공격 -2, 아군 회피 +3 |

## 원소 상성 vs 종족 시너지 관계

두 시스템이 겹치므로 편성 시 **트레이드오프** 발생:

```
전략 A: 종족 시너지 집중
  → 야수 4체 (야수 시너지 최대) + 원소 제각각
  → 시너지 보너스는 크지만, 원소 공명 없음

전략 B: 원소 공명 집중  
  → 화염 유닛 4체 (화염폭풍 발동) + 종족 제각각
  → 원소 보너스는 크지만, 종족 시너지 없음

전략 C: 밸런스 편성
  → 적 원소에 유리한 2~3체 + 종족 시너지 2체
  → 양쪽 적당히 챙기기
```

이 트레이드오프가 **편성의 전략적 깊이**를 만들어냄.

---

## 밸런스 고려사항

### 1.5배 vs 2배
- **1.5배 추천** (현재안): 상성이 중요하지만 결정적이지는 않음
- 2배로 하면 상성이 너무 지배적 → 편성 자유도 감소
- 업계 주요 카드게임들도 상성 배율을 1.3~1.5배로 유지

### 신성/암흑 밸런스
- 상호 상극이라 특정 상대에는 매우 유리하지만, 나머지 4원소에는 보통
- 신성/암흑이 너무 강해지지 않도록 "자신 외 원소에는 상성 없음" 유지

## 웹 구현 시 고려사항
- **ElementType**: `'fire' | 'water' | 'lightning' | 'earth' | 'dark' | 'holy'` 문자열 유니온
- **상성 테이블**: `const ELEMENT_ADV = { fire: { water: 0.75, earth: 1.5 }, ... }` 객체
- **VFX**: 유리/불리 타격 시 Canvas 2D 파티클 (화염=불꽃 파편, 빙결=얼음 파편 등)
- **UI**: `<span class="damage-text element-advantage">` + CSS 글꼴/크기/색상 차등
