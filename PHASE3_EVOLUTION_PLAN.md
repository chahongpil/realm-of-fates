# Phase 3-7: 카드 진화(전직) 시스템 계획

> 목표: 유닛 성장의 핵심 컨텐츠. 명예 축적 → 분기 선택 → 전직으로 유닛 특성 변환.

---

## 시스템 개요

### 전직 조건
- **레벨**: Lv.10 이상
- **명예**: 50 이상 (유닛별 개별 축적)
- **골드**: 전직 비용 (등급별 상이)

### 전직 규칙
- 전직 시 **2갈래 분기** 중 택 1 (돌이킬 수 없음)
- 전직하면: 이름/아이콘 변경, 스탯 대폭 상승, 새 스킬 획득, 등급 1단계 상승
- 전직 후에도 추가 성장 가능 (2차 전직은 Phase 4+)

### 전직 연출
1. 카드가 화면 중앙으로 이동
2. 빛의 기둥이 카드를 감쌈 (파티클)
3. 카드가 뒤집히며 새 일러스트 공개
4. 등급 상승 이펙트 (금빛 폭발)
5. 새 스킬 아이콘 + 이름 표시
6. 전용 사운드 (웅장한 팡파레)

## 확정 사항

### Q1: 전직 분기 수 → 3분기 허용 확정
> 기본은 2분기, 일부 유닛은 3분기 허용.
> 예: 견습마법사 → 화염술사 / 빙결술사 / **번개술사**
> 3분기 대상 유닛 추가 예시:
> - 민병 → 근위병 / 검투사 / **용병** (경제 특화)
> - 사냥꾼 → 저격수 / 수렵꾼 / **조련사** (야수 소환)
> - 늑대 → 다이어울프 / 망령늑대 / **알파울프** (무리 극대화)

### Q2: 2차 전직 조건 유지
> 골드→전설: Lv20 + 명예200 + 80골드 (현재안 유지)

### Q3: 경제 특화 전직 → 확정
> 해적(도적 전직) "골드 추가 획득" 방향 채택.
> 전투 승리 시 +2~3💰 보너스, 과도하지 않은 수준으로 조정.
