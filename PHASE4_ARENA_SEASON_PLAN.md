## Part A: 아레나 시스템

### 1. 실시간 PvP (1v1)

#### 기본 구조
```
매칭 → 편성(30초) → 실시간 오토배틀 → 결과

- 양쪽 플레이어가 동시에 유닛/스킬/유물 편성
- 편성 완료 후 오토배틀 시작
- 라운드 사이 선택지도 동시 진행 (타이머 15초)
- 상대방의 선택은 보이지 않음
```

#### 매칭 시스템
```
매칭 기준:
  1차: 아레나 점수 ±200점 범위
  2차: 30초 경과 시 범위 확장 (±400)
  3차: 60초 경과 시 AI 봇 매칭 (대기 시간 제한)

매칭 화면:
╔═══════════════════════════════╗
║     ⚔️ 영웅을 찾는 중...       ║
║                               ║
║   👤 나         VS    👤 ???  ║
║  ⭐ Lv.15              ???    ║
║  💎 다이아              ???    ║
║                               ║
║   🔍 매칭 중... 12초          ║
║   [취소]                       ║
╚═══════════════════════════════╝
```

#### 아레나 점수 (리그와 별개)
| 등급 | 점수 | 아이콘 |
|------|------|--------|
| 도전자 | 0~999 | ⚔️ |
| 투사 | 1000~1999 | 🗡️ |
| 용사 | 2000~2999 | 🛡️ |
| 영웅 | 3000~3999 | ⭐ |
| 전설 | 4000~4999 | 👑 |

| 승리 | +25 | +5~+15 |

### 2. 협동 PvE (길드 레이드)

#### 레이드 보스 시스템
```
길드원 여러 명이 동시에 거대 보스에 도전

보스 특징:
- HP가 극도로 높음 (길드원 전체가 합산 피해)
- 페이즈별 패턴 변경
- 각자 자기 덱으로 공격 → 데미지 합산
- 실시간 동시 접속 필요 없음 (비동기도 가능)
```

#### 레이드 보스 목록
| 보스 | 원소 | HP | 특수 패턴 | 보상 |
|------|------|-----|----------|------|
| 🐉 고대 드래곤 | 화염 | 50만 | 3페이즈마다 전체 브레스 | 전설 장비 |
| 🕷️ 거미여왕 | 암흑 | 30만 | 새끼거미 소환, 독 | 전설 스킬 |
| 🗿 타이탄 골렘 | 대지 | 80만 | 방어 극대화, 지진 | 신급 장비 |
| ❄️ 빙하의 군주 | 물 | 40만 | 전체 빙결, 분신 | 전설 유물 |
| ⚡ 뇌신의 사도 | 번개 | 60만 | 연쇄 번개, 속도 극대화 | 신급 스킬 |
| 👼 타락 천사 | 신성/암흑 | 100만 | 힐+공격 전환, 부활 | 신급 유닛 선택권 |

#### 레이드 보상 분배
```
개인 보상: 기여 데미지에 비례
길드 보상: 보스 처치 시 길드 전원에게 지급
MVP 보상: 최다 데미지 기여자 추가 보상
```

### 3. 협동 PvP (2v2)

#### 기본 구조
```
2명 팀 vs 2명 팀

편성 규칙:
- 각자 영웅 + 유닛 4체 (총 팀 8~10체)
- 팀원의 편성을 볼 수 있음
- 시너지를 맞추면 팀 시너지 보너스

전투:
- 아군 유닛이 합쳐져서 한 전장에서 싸움
- 팀원과 대화 가능 (간단한 핑/이모티콘)
```

#### 팀 시너지 보너스
- 두 플레이어의 유닛 종족이 겹치면 시너지 강화
- 예: A가 야수 2체, B가 야수 2체 → 야수 4체 시너지 발동!

#### 2v2 보상
- 일반 PvP보다 보상 1.5배
- 팀 전용 칭호 ("최강의 듀오" 등)

### 시즌 주기 (v2 2026-04-22)

### 시즌 리셋 규칙

#### 리그 점수 소프트 리셋
```
새 시즌 점수 = 이전 시즌 점수 × 50%

예시:
  신의영역 2500점 → 1250점 (다이아 시작)
  마스터 1800점 → 900점 (플래티넘 시작)
  다이아 1200점 → 600점 (플래티넘 시작)
  골드 400점 → 200점 (실버 시작)
  실버 150점 → 75점 (브론즈 시작)
```

#### 아레나 점수 하드 리셋
```
모든 플레이어 → 0점 (도전자)부터 다시 시작
```

#### 시즌 한정 콘텐츠
- **한정 칭호**: 해당 시즌에만 획득 가능 (예: "시즌1 정복자")
- **한정 코팅**: 시즌 테마 코팅 (예: 시즌1=화염, 시즌2=빙결...)
- **한정 프로필 프레임**: 마스터+ 달성자 전용

### 시즌 테마 (v2 2026-04-22 — 연 4 시즌 기준 재배치)

연 4 시즌 × 6 원소 = **1.5 년 (6 시즌) 1 사이클**. 3 달 시즌의 호흡에 맞춰 한 시즌당 1 원소 테마.

| 시즌 | 테마 | 특별 규칙 | 테마 신 |
|------|------|----------|--------|
| 1 | 🔥 불의 시즌 | 불 원소 공격 +10% | 그라힘 주간 (획득 확률 ↑) |

## Part C: 네트워크 구현

### 실시간 PvP 통신 흐름
```
[Player A] ←→ [Photon Fusion Server] ←→ [Player B]

1. 매칭 요청 → PlayFab Matchmaking
2. 매칭 성공 → Photon Room 생성
3. 편성 동기화 → 양쪽 편성 완료 신호
4. 전투 시작 → 서버에서 시드값 공유
5. 오토배틀 → 동일 시드로 양쪽 동일 결과 (결정론적 시뮬레이션)
6. 라운드 선택 → 각자 선택 → 서버에서 수집 → 동시 적용
7. 결과 → PlayFab에 점수 기록
```

### 결정론적 전투 (Deterministic Battle)
```
핵심: 같은 시드 + 같은 입력 → 같은 결과

방법:
- 전투 시작 시 랜덤 시드를 서버에서 공유
- 양쪽 클라이언트가 동일한 시드로 전투 시뮬레이션
- 결과가 항상 동일하므로 전투 자체를 동기화할 필요 없음
- 라운드 사이 선택만 동기화하면 됨
- 대역폭 최소화 + 치트 방지
```
