# Phase 4-A: 가챠 & 과금 시스템 계획

> 목표: 시즌패스 + 코스메틱(A안) + 가챠 천장(B안) 혼합 모델
> 마지막 업데이트: 2026-04-11

---

## 과금 철학

- **P2W 최소화**: 과금으로 "시간 단축"은 가능, "스탯 압도"는 불가
- **천장(Pity) 필수**: 일정 횟수 이상 뽑으면 확정 보상
- **코스메틱 중심**: 카드 코팅, 칭호, 프로필 장식 등 시각적 보상
- **시즌패스**: 매월 정액 구독으로 안정적 혜택

## 1. 가챠 시스템

### 유닛 가챠 (선술집)

#### 확률표
| 등급 | 확률 | 단가 |
|------|------|------|
| 브론즈 | 50% | 5💰 |
| 실버 | 30% | 5💰 |
| 골드 | 15% | 5💰 |
| 전설 | 4% | 5💰 |
| 신 | 1% | 5💰 |

#### 천장(Pity) 시스템
| 횟수 | 보장 |
|------|------|
| 10연차 | 실버 이상 1체 확정 |
| 50연차 | 전설 1체 확정 (소프트 피티: 40연차부터 전설 확률 +2%/회) |
| 100연차 | 신 1체 확정 (하드 피티) |

#### 소프트 피티 상세
```
1~39연차: 기본 확률 (전설 4%, 신 1%)
40~49연차: 전설 확률 매 회 +2% (4%→6%→8%...→24%)
50연차: 전설 확정 (피티 리셋)
80~99연차: 신 확률 매 회 +1% (1%→2%→3%...→21%)
100연차: 신 확정 (피티 리셋)
```

#### 10연차 보너스
- 10연차 1회 = 골드 50💰 (단차 5💰 × 10 = 50💰이지만, 10연차 패키지는 45💰)
- 10연차 시 추가 보너스: 랜덤 스킬 1개

## 2. 과금 상품

### 화폐 정리 (전 기획서 공용)
> ⚠️ 용어 혼란 방지:
> - **골드 (💰)** = 기본 화폐 (전투/퀘스트로 획득)
> - **보석 (💎)** = 프리미엄 화폐 (과금/시즌패스)
> - **축복 (✨)** = 특수 화폐 (연승/업적으로 획득, 특별 상품 구매)
> - "신의 축복"이라는 단어는 **축복 화폐**를 가리킴. 코팅 4등급은 **"신화"**임.

## 3. 경제 순환 구조

```
과금 ($) → 💎보석
           ↓
         ┌─────────────────────┐
         │  💎 → 💰골드         │
         │  💎 → 가챠 (유닛/스킬/장비) │
         │  💎 → 신의축복       │
         │  💎 → 편의 (갱신/체력) │
         └─────────────────────┘
         
무과금 루트:
  전투 승리 → 💰골드 → 가챠/건설/단련
  전투 승리 → 신의축복 → 특별 상품
  퀘스트 → 💰골드 + 스킬/유물
  업적 → 💰골드 + 칭호 + 코팅
  시즌 무료트랙 → 보상
```

### 무과금 vs 과금 차이
| | 무과금 | 라이트 과금 ($5/월) | 중과금 ($30/월) |
|---|---|---|---|
| 월 가챠 횟수 | ~20회 | ~50회 | ~150회 |
| 전설 획득 | 월 1~2체 | 월 3~5체 | 월 10체+ |
| 리그 도달 | 플래티넘 (3개월) | 다이아 (2개월) | 마스터 (1개월) |
| 코팅 | 빛나는까지 | 황금까지 | 전체 |

> 핵심: 과금은 "빠르게" 도달, 무과금은 "천천히" 도달. 최종 도달점은 동일.
