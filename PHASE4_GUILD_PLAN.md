# Phase 4-E: 길드 시스템 계획

> 목표: 본격 길드 — 레이드, 길드 건물, 공동 보상, 채팅
> 마지막 업데이트: 2026-04-11

---

## 1. 길드 기본

### 생성/가입
- **길드 생성**: Lv.5+, 50💰 비용
- **길드 이름**: 2~12자
- **길드 아이콘**: 엠블럼 선택 (원소별 6종 + 기본 10종)
- **길드 정원**: 기본 20명, 길드 레벨에 따라 최대 50명
- **가입 조건**: 길드장이 설정 (자유가입 / 승인제 / 초대제)

### 길드 직위
| 직위 | 인원 | 권한 |
|------|------|------|
| 길드장 | 1명 | 모든 권한, 길드 해산 |
| 부장 | 최대 3명 | 승인/추방, 길드 건물 업그레이드 |
| 정예 | 제한 없음 | 레이드 개시 |
| 일반 | 제한 없음 | 참여만 가능 |

---

## 2. 길드 레벨 & 건물

### 길드 경험치
- 길드원 전투 승리 → 길드 XP +5
- 레이드 참여 → 길드 XP +20
- 퀘스트 완료 → 길드 XP +3

### 길드 레벨
| 레벨 | 필요 XP | 해금 |
|------|--------|------|
| 1 | 0 | 기본 기능, 채팅, 정원 20명 |
| 2 | 500 | 길드 상점, 정원 25명 |
| 3 | 1500 | 길드 레이드, 정원 30명 |
| 4 | 4000 | 길드 버프, 정원 40명 |
| 5 | 10000 | 길드 대전, 정원 50명, 길드 엠블럼 커스텀 |

### 길드 건물 (길드 마을)
길드에도 공동 건물이 있으며, 길드원 전체에게 버프를 제공:

| 건물 | 효과 | 업그레이드 자원 |
|------|------|----------------|
| 🏛️ 길드 본부 | 길드 레벨 기본 건물 | 길드 골드 |
| ⚔️ 전쟁의 간 | 전투 공격력 +1~5% | 길드 골드 |
| 🛡️ 수호의 탑 | 전투 방어력 +1~5% | 길드 골드 |
| 💰 금고 | 길드 골드 보관량 증가 | 길드 골드 |
| 📜 지혜의 서원 | 퀘스트 보상 +5~15% | 길드 골드 |
| 🔥 용광로 | 장비 제작 보너스 +5~15% | 길드 골드 |

### 길드 골드
- 길드원이 개인 골드를 기부 (자발적)
- 레이드 보상의 일부가 길드 골드로
- 길드 건물 업그레이드/유지에 사용

## 3. 길드 레이드 (상세)

> 아레나 계획서(PHASE4_ARENA_SEASON_PLAN.md)의 레이드 보스 참조

### 레이드 규칙
- 길드 Lv.3 이상에서 해금
- 주 2회 개시 가능 (화/토)
- 참여 시간: 레이드 개시 후 24시간 이내
- 각 길드원은 1일 3회 공격 가능 (각자 자기 덱으로)
- 보스 HP는 길드 인원에 비례 조정

### 보상
```
클리어 보상 (전원):
  - 💰 개인 골드 + 길드 골드
  - 레이드 전용 장비/스킬 (레이드에서만 획득)
  - 길드 XP
  - ⭐ 명예 +20 (참여 유닛 전체) ← 전직 준비에 핵심

MVP 보상 (최다 기여):
  - 추가 골드 ×2
  - ⭐ 명예 +10 추가
  - 한정 칭호 "레이드 MVP"

GvG 대전 보상:
  - 승리 시: ⭐ 명예 +15 (참여 유닛)
  - 패배 시: ⭐ 명예 +5
```

### 명예 획득 경로 종합 (전 기획서 통합)
| 출처 | 명예 | 비고 |
|------|------|------|
| 전투 승리 | +10 | 참여 유닛 전체 |
| 전투 패배 | +1 | 참여 유닛 전체 |
| 길드 레이드 | +20 | 참여 유닛 전체 |
| 길드 대전 (승) | +15 | 참여 유닛 |
| 길드 대전 (패) | +5 | 참여 유닛 |
| 교회 장례 | +명예 | 유닛 해제 시 |
| 훈련장 명예수련 | +시간비례 | Lv5 해금 |
| 퀘스트 "자선가" | +10 | 일일 퀘스트 |

## 4. 길드 대전 (GvG)

### 규칙
- 길드 Lv.5에서 해금
- 주간 토너먼트 (매주 일요일)
- 참여 신청한 길드끼리 자동 매칭
- 각 길드에서 대표 5명 선출
- 5v5 순차 대전 (3선승)

### 보상
| 결과 | 보상 |
|------|------|
| 승리 | 길드 골드 +200, 참가자 💰50 |
| 패배 | 길드 골드 +50, 참가자 💰15 |
| 주간 1위 길드 | 한정 길드 칭호 + 💎100/인 |

## 5. 길드 채팅

### Photon Chat 활용
- 길드 전체 채팅 (기본)
- 귓속말 (1:1)
- 레이드 채팅 (레이드 진행 시)
- 이모티콘/스티커 (게임 내 캐릭터)
- 퀵 핑: "도움 요청", "수고했어요", "레이드 참여!" 등

## 6. PlayFab 길드 구현

```
PlayFab Entity Groups API:
  - 길드 = Entity Group
  - 길드원 = Group Member
  - 직위 = Role (admin, member 등)
  - 길드 데이터 = Group Data (건물 레벨, 골드 등)

Photon:
  - 길드 채팅 = Photon Chat Channel (길드ID)
  - 레이드 = Photon Room (길드 전용)
```
