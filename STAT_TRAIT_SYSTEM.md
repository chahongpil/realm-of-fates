# 스탯 배분 & 특성 트리 시스템

> 레벨업 시 스탯 포인트 + 특성 포인트 투자
> 마지막 업데이트: 2026-04-11

---

## 1. 레벨 시스템

```
최대 레벨: 100
레벨업 시: 스탯 포인트 +1 (매 레벨)
특성 포인트: 5레벨마다 +1 (총 20포인트)

Lv1 → Lv100 = 스탯 99포인트 + 특성 20포인트
```

---

## 2. 스탯 포인트 배분

### 1포인트당 상승치
| 스탯 | 1포인트당 | 비고 |
|------|----------|------|
| ATK | +1 | |
| DEF | +1 | |
| SPD | +1 | |
| LUCK | +1 | |
| EVA | +1 | |
| RAGE | +1 | |
| MEVA | +1 | |
| **HP** | **+5** | 기본값이 높아서 |
| **NRG** | **+5** | 기본값이 높아서 |

### Lv100 예시 (불 근접 전사 영웅, 올 HP 투자)
```
기본: HP 50, ATK 4, DEF 1, SPD 1...
99포인트 전부 HP: HP 50 + 495 = 545
→ 극단적 탱커 빌드 (공격력은 기본만)
```

### Lv100 예시 (균형 투자)
```
HP 30포인트: +150 → HP 200
ATK 20포인트: +20 → ATK 24
DEF 15포인트: +15 → DEF 16
SPD 10포인트: +10 → SPD 11
NRG 10포인트: +50 → NRG 55
나머지 14포인트 자유 배분
→ 균형 빌드
```

## 3. 특성 트리

### 구조
```
5레벨마다 특성 포인트 1개 (Lv5, 10, 15... 100)
총 20포인트

3개 브랜치 중 선택 투자:
  ├── 🔴 공격 브랜치 (딜러 특화)
  ├── 🔵 방어 브랜치 (탱커 특화)
  └── 🟢 지원 브랜치 (유틸 특화)

각 브랜치 7단계 (7포인트 풀투자 가능)
20포인트로 1개 풀 + 1개 반 or 3개 균등
```

### 🔴 공격 브랜치
| 단계 | 특성 | 효과 |
|------|------|------|
| 1 | 날카로운 일격 | ATK +3 |
| 2 | 약점 감지 | 크리티컬 데미지 ×2 → ×2.5 |
| 3 | 연속 베기 | 일반공격 15% 확률 2회 타격 |
| 4 | 피의 갈증 | 처치 시 ATK +2 (전투 중) |
| 5 | 관통력 | DEF 30% 무시 |
| 6 | 폭발적 분노 | RAGE 축적 ×1.5배 |
| 7 | **처형자** | HP 20% 이하 적에게 데미지 ×2 |

### 🔵 방어 브랜치
| 단계 | 특성 | 효과 |
|------|------|------|
| 1 | 강철 피부 | DEF +3 |
| 2 | 생명력 | HP +30 |
| 3 | 반격 태세 | 피격 시 25% 확률 반격 |
| 4 | 불굴 | HP 30% 이하 시 DEF ×2 |
| 5 | 자연 회복 | hpReg +3 |
| 6 | 보호 본능 | 아군 수호 확률 25% → 50% |
| 7 | **불사신** | 전투 중 1회, 치명상에서 HP 1로 생존 |

### 🟢 지원 브랜치
| 단계 | 특성 | 효과 |
|------|------|------|
| 1 | 마나 친화 | NRG +20 |
| 2 | 신속 시전 | 스펠 코스트 -1 (최소 1) |
| 3 | 연쇄 마법 | 스펠 시전 시 20% 확률 추가 시전 |
| 4 | 마나 재생 | nrgReg +2 |
| 5 | 원소 친화 | 원소 유리 데미지 ×1.5 → ×1.8 |
| 6 | 콤보 마스터 | 연계/협동 콤보 확률 +20% |
| 7 | **궁극의 힘** | 궁극기(RAGE100) 데미지 ×2 + 아군 전체 버프 |

## 4. 카드 고유 특성 (자동 해금)

> 공통 트리(3브랜치)와 별개로, 카드마다 고유 특성 3개가 Lv10/30/50에 자동 해금
> 포인트 투자 불필요 — 레벨만 달성하면 자동 적용

### 구조
```
카드 성장 = 공통 트리 (포인트 투자) + 고유 특성 (자동 해금)

공통 트리: 전 카드 동일 (공격/방어/지원 3브랜치)
고유 특성: 카드마다 3개씩 다름 (Lv10/Lv30/Lv50)
```

### 영웅 카드 고유 특성 (원소별)

| 원소 | Lv10 | Lv30 | Lv50 |
|------|------|------|------|
| 🔥 불 | 화상 데미지 +20% | 화염 공격 시 스플래시 | 불사조의 혼: 사망 시 30% 부활 |
| 💧 물 | 빙결 확률 +10% | 치유량 +30% | 빙하의 갑옷: 피격 시 20% 빙결 반사 |
| ⚡ 번개 | SPD +3 | 공격 시 15% 추가 타격 | 번개 연쇄: 3체까지 데미지 전이 |
| 🌿 대지 | DEF +2, HP +10 | hpReg +2 | 대지의 축복: 턴 종료 시 보호막 +5 |
| 🌑 암흑 | 흡혈 10% | 처치 시 NRG 전회복 | 영혼 지배: 처치한 적 30% 아군화 |
| ✨ 신성 | 치유량 +20% | 보호막 부여량 +50% | 천상의 심판: 3턴마다 적 전체 고정 피해 5 |

### 모집 유닛 고유 특성 (카드별)

| 카드 | Lv10 | Lv30 | Lv50 |
|------|------|------|------|
| 민병 | 생존본능: HP 30%↓ DEF +3 | 전선 사수: 도발 1턴 획득 | 베테랑: ATK/DEF +5 |
| 수비병 | 방패숙련: 반격 확률 +15% | 철벽: 보호막 +5 매턴 | 난공불락: 물리 데미지 -30% |
| 사냥꾼 | 급소사격: LUCK +3 | 덫설치: 적 SPD -2 | 일격필살: 크리티컬 시 추가 관통 |
| 도적 | 은신강화: 은신 2턴 | 독날: 공격 시 30% 독 부여 | 그림자걸음: 매턴 20% 회피 추가 |
| 늑대 | 무리강화: 야수당 ATK +3→+4 | 울부짖음 자동: 매 3턴 | 알파각성: 야수 전체 ATK +3 |
| 견습마법사 | 마나절약: 코스트 -1 | 연쇄시전: 20% 추가 시전 | 마법폭주: 스펠 데미지 ×1.5 |
| 기사 | 도발강화: 도발 시 DEF +3 | 반격달인: 반격 데미지 100% | 성기사의 맹세: 아군 HP↓ 시 수호 |
| 광전사 | 피의분노: RAGE 축적 ×1.3 | 광기강화: 50%↓ ATK ×2.5 | 불멸의 분노: HP 1로 1회 생존 |
| 화염술사 | 화상강화: 화상 +1턴 | 이중화염구: AOE 2회 시전 | 화염폭풍: 전체 3턴 화상 |
| 암살자 | 급소달인: 크리 확률 +15% | 은신처치: 처치 시 은신 확정 | 그림자군주: 은신 중 ATK ×2 |
| 성기사 | 신성치유강화: +50% | 신성보호막: 아군 전체 보호막 | 신성심판: 마법 데미지 무효 1회 |
| 죽음의기사 | 흡혈강화: 흡혈 80% | 사령술강화: 소환 50% | 죽음의지배: 언데드 ATK +5 |
| 드래곤 | 브레스강화: AOE +3 | 용의 비늘: DEF +5 | 태고의 분노: 전체 데미지 ×2 |
| 대천사 | 치유강화: 전체 HP +5/턴 | 천상의 보호: 아군 1회 무효 | 신의 심판: 적 전체 고정 10 |
| 리치 | 흡혈강화: 100% 회복 | 사령술강화: 소환 70% | 죽음의 왕: 사망 적 전부 소환 |
| 타이탄 | 번개강화: 연쇄 2체 | 뇌신의 갑옷: 피격 시 반격 번개 | 종말의 번개: 전체 ×3 데미지 |
| 불사조 | 부활강화: HP 70%로 부활 | 화염오라: 공격자 피해 +3 | 영원의 불꽃: 부활 무제한 |

> 카드 추가 시 고유 특성 3개만 정의하면 됨 (패턴화 가능)

## 5. 특성 리셋

```
리셋 비용: ✨축복 5개
  → 스탯 포인트 + 특성 포인트 전부 회수
  → 재배분 가능

무료 리셋:
  전직 시 1회 무료 리셋
  시즌 리셋 시 1회 무료 리셋
```
