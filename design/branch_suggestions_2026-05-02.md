## 컨셉 라인 정책 (이번 세션 결정)

- **B안 채택**: 컨셉 라인 자유 (원소·역할 섞임 가능)
- **소프트 제약**: 한 라인 = 1 원소 또는 친화 원소 2개 / 1 역할 위주 (단 진화 단계에서 역할 변화 1회 허용)
- 5 단계 = bronze → silver → gold → legendary → divine

## 추천안 — 6 신좌 직속 라인 (lore-bible 기반)

### 🔥 그라힘 직속 (불·attack)
**컨셉**: "재의 왕의 불꽃 검사 / 화염 망치 전사"
**소프트 제약**: 불 원소, attack 역할
- bronze: 화염병 / 검사 / 신병
- silver: 화염의 전사 (`flame_warrior`?)
- gold: 화염 검사단장 / 불꽃 사도
- legendary: 화염 처형자
- divine: **그라힘의 사도** (= 신 카드)

### 🌊 모라스 직속 (물·support+defense)
**컨셉**: "심연의 어머니 — 치유사·기억 사제 계열"
**소프트 제약**: 물 원소, support 역할 (defense 1단계 허용)
- bronze: 견습 사제 / 우물 지기
- silver: 물의 사제 (`water_priest`?)
- gold: 망각의 인도자
- legendary: 심연의 산파
- divine: **모라스의 3 머리 사도**

### ⛰️ 에이드라 직속 (땅·defense)
**컨셉**: "외눈의 수호자 — 키클롭스 형 방패병 계열"
**소프트 제약**: 땅 원소, defense 역할
- bronze: 돌 방패병 / 보초
- silver: 산악 수호자 / 대지의 수호자 (`earth_guardian`?)
- gold: 산악 파괴자 / 돌의 거인
- legendary: 외눈 수호장
- divine: **에이드라의 형 사도**

### ⚡ 브론테스 직속 (전기·attack+ranged)
**컨셉**: "분노의 망치 — 키클롭스 동생 번개 강타 계열"
**소프트 제약**: 전기 원소, attack 역할 (속도 + 선제공격 강조)
- bronze: 번개 활병 / 활잡이
- silver: 폭풍 궁수 (`storm_archer`?)
- gold: 천둥의 망치
- legendary: 번개 타이탄 (`lightning_titan`?)
- divine: **브론테스의 동생 사도**

### ✨ 세라피엘 직속 (신성·support+attack 하이브리드)
**컨셉**: "새벽의 창 — 빛의 기사·성기사 계열 (보호파 리더)"
**소프트 제약**: 신성 원소, support → attack 진화
- bronze: 견습 사제 / 빛의 신병
- silver: 빛의 기사 / 성당 기사
- gold: 성기사 (`paladin`?)
- legendary: 성기사단장 / 새벽 창
- divine: **세라피엘의 천사 사도** (= 보호파 리더 후계)

### 🌑 네크리온 직속 (암흑·attack+luck)
**컨셉**: "밤의 계약자 — 암살자·계약 마법사 계열 (말살파 리더)"
**소프트 제약**: 암흑 원소, attack 역할 (luck 특화)
- bronze: 그림자 도적 / 견습 계약자
- silver: 암살자 (`assassin`?)
- gold: 어둠 사도 / 거래의 손
- legendary: 밤의 계약자
- divine: **네크리온의 타락 사도** (= 말살파 리더 후계)

## 무파벌 / 하이브리드 라인 (선택)

### 🌪️ 폭풍 라인 (전기 + 물)
**컨셉**: "두 신의 경계 — 번개 폭풍 + 비"
- bronze: 빗속 궁수
- silver: 폭풍 마법사
- gold: 천둥비 사령관
- legendary: 폭풍의 화신
- divine: 두 신의 사도

### 🌋 화산 라인 (불 + 땅)
**컨셉**: "그라힘 + 에이드라 경계 — 용암"
- bronze: 화산 광부
- silver: 용암병
- gold: 마그마 거인
- legendary: 화산 군주
- divine: 화산의 화신

### 🦇 황혼 라인 (신성 + 암흑)
**컨셉**: "세라피엘 + 네크리온 경계 — 그림자 천사 / 빛의 악마"
**lore 정합**: 한때 동료였던 두 신의 경계 — 매우 희귀한 카드 라인
- bronze: 황혼의 신병
- silver: 그림자 사제
- gold: 황혼의 검사
- legendary: 빛-어둠 양면자
- divine: 분리 전 천사 (lore: 세라피엘·네크리온 분리 전 모습)

## 가지뷰 입력 가이드

`tools/units_flow_map.html` 띄움 → ② 가지뷰 탭 → "+ 새 라인 추가" 클릭

각 라인:
- 이름: 위 표의 라인명 (예: "그라힘 직속")
- 컨셉: 위 컨셉 텍스트 (예: "재의 왕의 불꽃 검사 / 화염 망치 전사")
- bronze 슬롯: 유닛 ID (예: `flame_warrior` 또는 placeholder 이름)
- silver/gold/legendary/divine: 같은 방식

LocalStorage 자동 저장 → 새로고침해도 유지. CSV 다운로드 가능.
