## 🛡️ 역할별 외곽 실루엣 규칙

프레임 PNG 의 **외곽 테두리 모양** 만 역할별로 달라지며, **내부 14개 영역 좌표는 30장 전부 동일**.

| 역할 | 외곽 실루엣 |
|---|---|
| **탱커** (`tank`) | 방패형 — 하단이 V 자로 뾰족. 상단은 수평. 두꺼운 리벳 테두리 |
| **DPS** (`dps`, 근접+원거리) | 표준 직사각 + 상단 고딕 아치. 샤프한 모서리 |
| **지원** (`support`) | 타원·둥근 모서리. 수정구·리본 장식. 부드러운 곡선 |

## 🎴 카드 상태별 스케일 (참조)

이 레이아웃 스펙은 **1024×1536 원본** 기준. 실제 렌더 시 다음 비율로 축소:

| 상태 | 비율 | 픽셀 크기 |
|---|---|---|
| 풀사이즈 (컬렉션/상세) | 100% → 이후 CSS 에서 추가 축소 | 350 × 525 (기존) |
| 전장 기본 (PHASE 3 v2) | ~17% | 172 × 258 |
| 전장 중앙 확대 (char-focus) | ~42% | 430 × 645 |
| 손패 | ~14% | 140 × 210 |
| 매칭 초상화 | ~20% | 200 × 300 |

CSS 변수(`--bv2-card-w/h`)가 스케일 담당. 레이아웃 스펙은 원본 기준 1:1.
