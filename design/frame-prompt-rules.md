# 🔒 프레임 프롬프트 고정 UI 규격

> 2026-04-18 대표님 지정 / 2026-04-18 1024×1536 확정. **절대 변경 금지.**
> AI 이미지 생성(Gemini/MJ/SD) 프롬프트 가이드 + CSS 오버레이 좌표 기준.

## 📐 캔버스 (확정)

| 항목 | 값 |
|---|---|
| **너비** | 1024 px |
| **높이** | 1536 px |
| 비율 | 2:3 (세로) |
| 포맷 | PNG-32 (중앙 투명 창) |

## 🎯 고정 UI 규격 — 두 가지 표기

좌측 "프롬프트용" 은 AI 이미지 생성(숫자 깔끔해야 복붙 편함), 우측 "CSS 환산값" 은 실제 렌더 오버레이 좌표.

### 1️⃣ 좌측 4개 스탯 슬롯

- **크기**: 150×80 **→ CSS: 124×66**
- **좌표 (좌상단 기준)**:
  | # | 색 | 스탯 | 프롬프트용 | CSS 환산 |
  |---|---|---|---|---|
  | 1 | 🔴 빨강 | ATK | (160, 520) | (132, 430) |
  | 2 | 🔵 파랑 | DEF | (160, 610) | (132, 504) |
  | 3 | 🟢 초록 | SPD | (160, 700) | (132, 578) |
  | 4 | 🟡 노랑 | LUCK | (160, 790) | (132, 652) |
- **세로 간격**: 약 10 px (슬롯 높이 + 간격)
- **순서**: 빨 → 파 → 초 → 노 **절대 고정** (등급·역할 불문)

### 2️⃣ 상단 두루마리 이름표

- **크기**: 940×180 **→ CSS: 776×149**
- **좌표**: 프롬프트용 (210, 220) **→ CSS: (173, 182)**
- **규칙**:
  - 항상 중앙 정렬 느낌 유지
  - 프레임과 살짝 떨어져 있어야 함
  - **크기 절대 변경 금지**

### 3️⃣ 우상단 HP 루비 (빨강)

- **크기**: 150×150 **→ CSS: 124×124**
- **좌표**: 프롬프트용 (1080, 430) **→ CSS: (893, 355)**
- **규칙**:
  - **항상 빨강** — 등급·원소 불문
  - 위치 **절대 고정**
  - 다른 색으로 바꾸지 않음

### 4️⃣ 우하단 NRG 루비 (파랑 / 에너지)

- **크기**: 150×150 **→ CSS: 124×124**
- **좌표**: 프롬프트용 (1080, 1320) **→ CSS: (893, 1091)**
- **규칙**:
  - **항상 파랑** — 등급·원소 불문
  - 상단 HP 루비와 **완전 동일 크기**
  - 위치 **절대 고정**

## 💡 왜 두 숫자 체계를 병기하나

- **프롬프트용 숫자**는 대표님이 기억·수정하기 쉬운 라운드 넘버. AI 이미지 생성 엔진(Gemini 등) 은 정확한 픽셀 좌표를 이해하지 못하므로 어느 숫자를 써도 결과 동일.
- **CSS 환산값**은 실제 1024×1536 캔버스에 오버레이를 얹을 때의 정확한 픽셀. 프론트엔드 구현 시 이 쪽을 사용.
- 배율 = 1024/1240 ≈ 0.8258 (대표님 원본 기준 축소).

```
절대 다른 해상도(1200, 1240, 1500 등)로 바꾸지 마.
```
