## 2. 등급 업그레이드 효과

### 2.1 공통 규칙

- **스탯 배수**: `atk · hp · maxHp · def` × 1.5 (04-balance.md § 진화 계수 이미 존재)
- **속도 계열**: `spd · nrg · luck · eva` × 1.3 (동)
- **rarity**: 한 단계 상승
- **고유 능력 자동 추가**: 업그레이드마다 1 개 (아래 § 2.2)
- **레벨 유지**: Lv 50 bronze → silver 업그레이드 후 **Lv 50 silver** (투자 보호)

### 2.2 등급별 자동 추가 능력 (예시 — 실 카드별 차등)

| 업그레이드 경로 | 자동 추가 능력 방향 | 예시 |
|---|---|---|
| bronze → silver | **소형 패시브** (관통·반격 미량) | "관통 +1" / "피격 시 1 회 반격" |
| silver → gold | **분기 선택 1 회 + 중형 패시브** | 공격형: "첫 타격 광역" / 방어형: "카운터 반격" |
| gold → legendary | **중대 패시브** (처형·흡혈·재생) | "처형 (HP 30% 이하 적 즉사)" / "흡혈 20%" |
| legendary → divine | **신 전용 고유 능력** | "무한 추격 (적 처치 후 추가 행동)" / "영혼 연쇄" |

- 각 카드 ID 별 능력은 `js/11_data_units.js` 의 `upgradeEffects: {silver:'...', gold:'...'}` 필드로 정의 (신규 스키마).

### 2.3 희귀 → 고귀한 분기 상세

- **선택지 A (공격형)**: "첫 타격 광역" / "치명타 +15" / "연속 공격" 등
- **선택지 B (방어형)**: "카운터 반격" / "DEF +5" / "피해 감소 10%" 등
- **1 회만 선택** — 되돌리기 없음
- **전직 초기화권** (✨ 10) 으로 재선택 가능 (`CURRENCY_SYSTEM.md` § 4)
- 다른 등급 업그레이드 (일반→희귀, 고귀한→전설의, 전설의→신) 는 **분기 없음, 자동 추가만**

## 3. 경험치 곡선

```
xp_to_next_level(L) = 100 × L^1.5     ← 기존 04-balance 식 유지
```

**단계별 XP 예시**:
- Lv 1 → 2: 100 XP
- Lv 10 → 11: ≈ 3,162 XP
- Lv 50 → 51: ≈ 35,355 XP
- Lv 99 → 100: ≈ 98,499 XP

**일반 (bronze Lv 60) 플레이어의 고귀한 (gold Lv 80) 도달 경로**:
1. bronze Lv 1 → 60 = 약 45 만 XP
2. bronze → silver 업그레이드 (재료 소모, Lv 60 유지)
3. silver Lv 60 → 70 = 약 13 만 XP
4. silver → gold 업그레이드 + **분기 선택** (Lv 70 유지)
5. gold Lv 70 → 80 = 약 13 만 XP

합계: 약 71 만 XP (일반~고귀한 풀 레벨)

## 5. 등급 업그레이드 재료

| 업그레이드 | 재료 구성 |
|---|---|
| 일반 → 희귀 | 중복 카드 2 장 + 골드 (소량) |
| 희귀 → 고귀한 | 중복 카드 3 장 + 신의 잔재 소량 + **분기 선택 화면** |
| 고귀한 → 전설의 | 중복 카드 4 장 + 신의 잔재 중량 |
| 전설의 → 신 | **시즌 보상으로만 획득** (`monetization.md` § 3.1/3.2/3.4 — 조각 누적 완성) — 합성 불가 |

## 6. 출시 범위 vs 후속 업데이트

### 출시 범위 (이번 결정 포함)

- Lv 100 + 등급별 상한 60/70/80/90/100 고정
- 등급 업그레이드 자동 능력 추가 (예시 4 건 + 카드별 매핑)
- 희귀 → 고귀한 1 회 분기 선택 (공격형/방어형)
- 전설의 → 신 합성 불가 (시즌 보상 전용)

### 후속 업데이트

- **전설 → 신 2 차 분기** (옵션) — 시즌 3~5 에 추가 고려
- **다중 분기 트리** (스킬 Lv 내부 깊이) — 성장 깊이 확장
- **분기별 전용 카드 아트/스킨** — 공격형·방어형 시각 차등
