## 검증된 동작

### 1. 비전투 화면 (Tavern/Deckview/Castle)
- `.card-v4 .bars { display: none }` 규칙 정상 동작 → 바 0% 노출
- `.top-row { top: 8px }` → 이름/Lv 박스 최상단 위치
- `.name-box` 에 이름 + HP 숫자 (♥ NNN) 정상 표시
- `.lv-box` 에 Lv + NRG 숫자 (◆ N) 정상 표시
- 캐릭터 일러스트 얼굴/투구 완전히 노출 (대표님 리디자인 목표 달성)

### 2. 전투 컴팩트 (DOM 주입 테스트)
- `.card-v4-compact` 클래스 시 바 자동 표시 (`display: flex`)
- `.top-row` 가 자동으로 `top: 28px` 로 밀림
- HP/NRG 바 + 라벨 (`HP 50 / 80`, `NRG 0 / 20`) 정상
- 이름박스 + Lv박스도 유지 → 이중 정보 (바 + 숫자) 로 전투 중 즉시 파악 가능

### 3. Setter API 동기화
- `setHP(n)` → 바 fill + 라벨 + 이름박스 내 `♥ n` 3곳 동시 갱신
- `setNRG(n)` → 바 fill + 라벨 + Lv박스 내 `◆ n` 3곳 동시 갱신
