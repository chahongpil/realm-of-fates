## 🧠 맥락 복구 노트 (다음 Claude 가 꼭 기억할 것)

1. **이 세션은 트랙 5 에서 범위 밖 작업이 대거 발생** — 대표님 직접 지시로 js/css/img 를 연속 편집. 커밋 범위 구분시 주의 필요.

2. **letterbox 3회 번복** — 대표님이 letterbox 자체를 싫어하심. 최종 해결은 "이미지를 1280×720 native 로 맞추기". 다음에 aspect 문제 나올 때 "image vs stage aspect" 계산을 먼저 확정하고 건드릴 것.

3. **"빨간색 연결됨" 의 진짜 정체** — 버튼 텍스트가 아니라 `.town-footer .btn` 의 battleButtonPulse + 빨간 그라디언트 스타일 오남용. 비슷한 "과한 CTA" 오류가 다른 화면에도 있을 수 있으니 검수 포인트로 기억.

4. **영상 배경 재적용 시 파일 크기 주의** — 원본 10.9MB → CRF 28 으로 902KB. 화질은 미세 애니용으로 충분. 만약 더 선명 필요시 CRF 22~23 (2~3 MB) 까지 올릴 수 있음.

5. **건물 좌표 % = container 기준 중심점** — editor_v2 와 game 이 동일 규약. 좌표 복사/붙여넣기 시 anchor 변환 불필요.

6. **initBuildings 는 모든 건물 Lv1 강제** — 기존 save 의 lv:0/null 도 1로 승격. "처음부터 다 지어져 있음" 이 기본. 원복하려면 initBuildings 로직 되돌려야 함.

7. **temple / inn action 미구현** — 클릭시 다이얼로그는 뜨지만 "들어가기" no-op. 구현 전까지 UX 흐름이 끊김.

8. **`lvNames` 데이터는 살아있음** — 표시만 중단. 추후 다시 레벨별 이름 쓸 수 있게 유지.
