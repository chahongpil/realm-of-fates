## 📝 이번 세션의 주요 결정

### 2026-04-27 오후

- **자원 관리 게임 방향 전환** (StS 스타일): 풀 충전 자동화 폐기 → currentHp/curNrg 영구 누적 소모. NRG 회복=여관, 부상 회복=교회 분리.
- **trait 시스템 폐기**: 14종 정의했지만 자동 부여 라벨에 그쳐 게임 깊이 미생성 → 정리. 진짜 필요하면 패시브 스킬로 흡수.
- **deckview 시각 일관성 강화**: 텍스트 설명창 폐기 → 카드 클릭 = 카드 확대. 전투 char-focus 와 동형으로 통일.
- **자동 로그인 opt-in 자체 폐기**: opt-in 이 "한 번 켜면 영구" 구조라 사용자 의도와 어긋남. 매번 로그인 화면.
- **사운드 패널 단순화**: 호버 펼침 거슬림 + 컨트롤 분산 → 톱니 단일 + 설정창 통합.
- **타이틀 어둠 그라디언트 = 경계선 원인**: game-root 영역에만 적용되어 letterbox 와 brightness 차이 → 제거.
- **NPC 프롬프트 워크플로우**: `design/{용도}_{날짜}.txt` 저장 + `start "" 경로` 로 파일 OPEN. 메모리 규칙 정정 ("띄워줘" = 파일 열기).
