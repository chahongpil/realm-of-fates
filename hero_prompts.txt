═══════════════════════════════════════════════════════════════
  사용 팁
═══════════════════════════════════════════════════════════════

1. ChatGPT 사용 시:
   - "다음 프롬프트로 이미지를 생성해줘 (1024x1536 세로)" 먼저 입력
   - 그 다음 프롬프트 본문 복붙
   - 네거티브는 "피하라:" 라고 한국어로 요청해도 됨

2. Stable Diffusion (DreamShaper) 사용 시:
   - 프롬프트 본문 그대로 입력
   - 네거티브 섹션을 별도 negative prompt에 넣기
   - Sampling steps 25-30, CFG 7.5 추천

3. 저장:
   - 파일명은 id 그대로 (예: h_m_fire.png)
   - c:/work/game/img/ 폴더에 덮어쓰기

4. 품질 확인:
   - cd /c/work/game && node tools/game_inspect.js units-grid --full
   - shots/units-grid.png 열어서 프레임에 잘 맞는지 확인

5. 재생성 판단:
   - 얼굴이 프레임 밖으로 나감 → 재생성
   - 원소 톤이 약함 → 재생성
   - 전신 샷이 나옴 → 재생성 (chest-up만 필요)
