🎯 카드별 슬롯 좌표 에디터

대표님 합성 PNG 는 카드마다 슬롯(HP/NRG/이름표/스탯) 위치가 다릅니다.
1. 드롭다운 또는 ◀ ▶ 로 카드 선택 → 2. 스테이지에서 마커/박스 드래그 → 3. 저장 버튼.
Override 없는 카드는 등급 기본값을 시드로 보여줍니다. 수정 안 하면 저장해도 해당 카드는 override 없이 기본값 사용.
등급기본값 리셋 → 이 카드의 override 제거. 저장 시 영구 반영.
